# Furosemide: จากภาวะบวมน้ำและความดันโลหิตสูงสู่ความดันโลหิตสูงในหลอดเลือดไตชนิดร้ายแรง

## สรุปสั้นๆ

Furosemide เป็นยาขับปัสสาวะกลุ่ม loop diuretic ที่ใช้กันแพร่หลายในการรักษาภาวะบวมน้ำ ความดันโลหิตสูง และภาวะหัวใจล้มเหลว โดยออกฤทธิ์ผ่านการยับยั้งตัวขนส่ง NKCC2 ใน ascending limb ของ loop of Henle
โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **ความดันโลหิตสูงในหลอดเลือดไตชนิดร้ายแรง (Malignant Renovascular Hypertension)**
ปัจจุบัน **ไม่มีการทดลองทางคลินิกที่ลงทะเบียนโดยตรง** แต่มี **วรรณกรรม 14 ฉบับ** ที่เกี่ยวข้อง ส่วนใหญ่สะท้อนบทบาทของ Furosemide ในการวินิจฉัยและการศึกษา RAAS ในผู้ป่วยความดันโลหิตสูงทุติยภูมิ

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียดใน Evidence Pack อย่างไรก็ตาม Furosemide เป็นที่ทราบกันดีว่าออกฤทธิ์ยับยั้งตัวขนส่ง NKCC2 (Na⁺-K⁺-2Cl⁻ cotransporter) ในส่วน thick ascending limb ของ loop of Henle ทำให้เกิดการขับโซเดียม คลอไรด์ และน้ำส่วนเกินออกทางปัสสาวะอย่างมีประสิทธิภาพ ผลลัพธ์คือปริมาตรเลือดหมุนเวียนลดลงและความดันโลหิตลดลงตามมา

ความเชื่อมโยงระหว่าง Furosemide กับ malignant renovascular hypertension มีสองมิติหลัก: ประการแรก Furosemide ถูกใช้มายาวนานในฐานะ "furosemide stimulation test" เพื่อกระตุ้นการหลั่ง renin ช่วยวินิจฉัยแยกแยะความดันโลหิตสูงทุติยภูมิจากหลอดเลือดไตออกจากชนิดอื่น (เช่น primary aldosteronism) ประการที่สอง ในทางทฤษฎีการลดปริมาตรเลือดอาจช่วยบรรเทาความดันโลหิตที่สูงมากในช่วงวิกฤต

อย่างไรก็ดี มีข้อกังวลสำคัญด้านกลไกที่ต้องคำนึงถึง: ในผู้ป่วย malignant renovascular hypertension ที่มีการตีบแคบของหลอดเลือดแดงไตอยู่แล้ว การออกฤทธิ์ลดปริมาตรเลือดของ Furosemide อาจกระตุ้น RAAS ให้รุนแรงขึ้นอีก ส่งผลให้การไหลเวียนเลือดในไตแย่ลงและไตเสียหายมากขึ้น ซึ่งสอดคล้องกับ mechanistic rationale ที่ระบุใน Evidence Pack ว่า "การออกฤทธิ์ลดปริมาตรเลือดอาจกระตุ้น RAAS และนำไปสู่ไตเสียหาย"

## ข้อมูลการวางจำหน่ายในประเทศไทย

ไม่พบใบอนุญาตการขึ้นทะเบียน Furosemide ในฐานข้อมูล TFDA จากการสืบค้น ณ วันที่ 25 มีนาคม 2569 (สถานะ: "未上市") ทั้งนี้ Furosemide เป็นยาในบัญชี WHO Essential Medicines และมีการใช้กว้างขวางทั่วโลก ขอแนะนำให้ตรวจสอบฐานข้อมูล TFDA โดยตรงเพื่อยืนยันสถานะการขึ้นทะเบียน

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
หลักฐานปัจจุบันอยู่ในระดับ L4 เท่านั้น ไม่มีการทดลองทางคลินิกที่ลงทะเบียนโดยตรง และวรรณกรรมที่มีส่วนใหญ่สะท้อนบทบาทการวินิจฉัยของ Furosemide (stimulation test) ไม่ใช่หลักฐานการรักษา malignant renovascular hypertension นอกจากนี้ยังมีข้อกังวลด้านกลไกว่าอาจกระตุ้น RAAS ให้รุนแรงขึ้นในผู้ป่วยกลุ่มนี้

**หากต้องการดำเนินการต่อต้อง:**
- ดาวน์โหลดและวิเคราะห์ข้อมูลกลไกการออกฤทธิ์ (MOA) และข้อห้ามใช้จาก DrugBank API
- ดาวน์โหลดและวิเคราะห์คำเตือนและข้อห้ามใช้จากเอกสารกำกับยาของ TFDA
- ตรวจสอบสถานะการขึ้นทะเบียน Furosemide ใน TFDA โดยตรง เนื่องจากเป็นยาหลักสากลที่ควรมีข้อมูลการจดทะเบียน
- ชี้แจงว่าการทำนายของ TxGNN เกี่ยวข้องกับบทบาทการวินิจฉัยหรือการรักษาของ Furosemide ในภาวะนี้ เพื่อกำหนดทิศทางการวิจัยที่ถูกต้อง
- หากต้องการประเมินต่อ ควรออกแบบการศึกษาก่อนคลินิกที่ประเมินผล furosemide ต่อ renovascular hypertension model โดยตรง โดยวัดผลต่อ renal perfusion และการกระตุ้น RAAS ควบคู่กัน
