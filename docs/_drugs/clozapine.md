# Clozapine: จากโรคจิตเภทต้านการรักษาสู่ความผิดปกติทางอารมณ์แบบไบโพลาร์ระยะคลั่ง

## สรุปสั้นๆ

Clozapine เป็นยาต้านโรคจิตชนิดผิดปกติ (atypical antipsychotic) ที่ได้รับการยอมรับในระดับสากลให้เป็น "มาตรฐานทอง" สำหรับการรักษาโรคจิตเภทที่ดื้อต่อการรักษา (treatment-resistant schizophrenia)
โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **ความผิดปกติทางอารมณ์แบบไบโพลาร์ระยะคลั่ง (Manic Bipolar Affective Disorder)**
ปัจจุบันมี **การทดลองทางคลินิก 6 รายการ** และ **วรรณกรรม 20 ฉบับ** สนับสนุนแนวทางนี้ รวมถึง meta-analysis และการทดลองแบบ double-blind ที่ประเมิน Clozapine ในโรคไบโพลาร์โดยตรง

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

Clozapine ออกฤทธิ์ผ่านการปิดกั้นตัวรับหลายชนิดพร้อมกัน ได้แก่ โดปามีน D2/D4 เซโรโทนิน 5-HT2A ฮิสตามีน H1 และอัลฟา-1 อะดรีเนอร์จิก กลไกหลายตัวรับ (multi-receptor antagonism) นี้เป็นพื้นฐานของประสิทธิภาพที่เหนือกว่ายาต้านโรคจิตชนิดอื่น และเป็นสิ่งที่ยา atypical antipsychotic รุ่นหลังหลายตัวพยายามเลียนแบบ

ในโรคไบโพลาร์ระยะคลั่ง (manic phase) การส่งสัญญาณโดปามีนที่มากเกินไปในวงจรลิมบิกและสไตรเอตัมเชื่อมโยงโดยตรงกับอาการคลั่ง ฉะนั้นการที่ Clozapine ปิดกั้นตัวรับ D2/D4 ได้อย่างกว้างขวางจึงทำให้เกิดฤทธิ์ปรับสมดุลอารมณ์ ทั้งโรคจิตเภทและโรคไบโพลาร์มีการทับซ้อนกันในแง่พยาธิสรีรวิทยาของระบบโดปามีน-เซโรโทนิน และผู้ป่วยบางส่วนที่มีอาการทางจิตในระยะคลั่งก็ถูกรักษาด้วย Clozapine มาโดยตลอดในทางปฏิบัติ

น้ำหนักของหลักฐานสนับสนุนสมมติฐานนี้ในระดับ L2 ได้แก่ systematic review และ meta-analysis (PMID 32182485) ที่ประเมินประสิทธิผลของ Clozapine ในโรคไบโพลาร์โดยตรง และการทดลองทางคลินิก Phase 2 แบบ double-blind ที่เสร็จสิ้นแล้ว (NCT00029458) ซึ่งตรวจสอบ Clozapine เฉพาะในโรคไบโพลาร์ระยะคลั่งที่ดื้อต่อการรักษา อย่างไรก็ตาม ยังขาด Phase 3 RCT ที่เสร็จสมบูรณ์สำหรับข้อบ่งใช้นี้โดยเฉพาะ

**การตัดสินใจ: Proceed with Guardrails**

**เหตุผล:**
มีการทดลองทางคลินิก Phase 2 แบบ double-blind ที่เสร็จสิ้นแล้วซึ่งตรวจสอบ Clozapine โดยตรงในโรคไบโพลาร์ระยะคลั่งที่ดื้อยา ประกอบกับ meta-analysis และ systematic review ที่แสดงประสิทธิผล อย่างไรก็ตาม ยาไม่ได้วางจำหน่ายในประเทศไทย และยังขาดข้อมูลคำเตือน/ข้อห้ามใช้ฉบับไทย รวมถึงกลไกการออกฤทธิ์โดยละเอียดที่ยืนยันจากฐานข้อมูลสากล

**หากต้องการดำเนินการต่อต้อง:**
- ดาวน์โหลดและวิเคราะห์เอกสารกำกับยา (SmPC/PIL) จาก EMA, FDA หรือ TGA เพื่อรับข้อมูลคำเตือนและข้อห้ามใช้อย่างครบถ้วน (แก้ไข DG001)
- สืบค้นกลไกการออกฤทธิ์โดยละเอียดจาก DrugBank API (แก้ไข DG002) เพื่อวิเคราะห์ความสัมพันธ์เชิงกลไกกับโรคไบโพลาร์อย่างเป็นระบบ
- ประเมินแผนการติดตามความปลอดภัยสำหรับผลข้างเคียงที่ทราบจากการใช้ในโรคจิตเภท เช่น ภาวะเม็ดเลือดขาวต่ำรุนแรง (agranulocytosis) หัวใจอักเสบ (myocarditis) อาการชัก และน้ำหนักตัวเพิ่มอย่างมีนัยสำคัญ
- พิจารณาขอจดทะเบียนยา (named patient program หรือ compassionate use) กับ TFDA หากวางแผนนำมาใช้ในบริบทของประเทศไทย
