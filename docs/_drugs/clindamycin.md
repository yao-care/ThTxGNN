# Clindamycin: จากการติดเชื้อแบคทีเรียสู่กระจกตาอักเสบแบบจุด

## สรุปสั้นๆ

Clindamycin เป็นยาปฏิชีวนะกลุ่ม Lincosamide ที่ใช้รักษาการติดเชื้อแบคทีเรียกลุ่ม Gram-positive และแบคทีเรียไม่ใช้ออกซิเจน โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **กระจกตาอักเสบแบบจุด (Punctate Epithelial Keratoconjunctivitis)** เป็นข้อบ่งใช้ใหม่อันดับแรก อย่างไรก็ตาม ปัจจุบัน**ไม่มีการทดลองทางคลินิกหรือวรรณกรรมสนับสนุนโดยตรง** การทำนายนี้จึงอิงเฉพาะโมเดล AI เท่านั้น

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์ (MOA) ในระบบอย่างเป็นทางการ แต่จากฐานความรู้เภสัชวิทยาสากล Clindamycin เป็นยาปฏิชีวนะกลุ่ม Lincosamide ที่ยับยั้งการสังเคราะห์โปรตีนของแบคทีเรียโดยจับกับ ribosomal subunit 50S ออกฤทธิ์ต้านเชื้อ Gram-positive (S. aureus, Streptococcus) และแบคทีเรียไม่ใช้ออกซิเจนหลายชนิด

กระจกตาอักเสบแบบจุด (Punctate Epithelial Keratoconjunctivitis) มีสาเหตุหลักจากไวรัส ความเป็นพิษของยา หรือปฏิกิริยาภูมิแพ้ ไม่ใช่การติดเชื้อแบคทีเรียโดยตรง Clindamycin อาจมีบทบาทจำกัดในทางทฤษฎีต่อการป้องกันการติดเชื้อแบคทีเรียแทรกซ้อน แต่ไม่ตรงกับกลไกหลักของโรค ตามที่ระบุใน Evidence Pack ว่า "แบคทีเรียเป้าหมายไม่ชัดเจน และปัจจุบันไม่มีหลักฐานทางคลินิกหรือวรรณกรรมใดรองรับ"

การทำนายของ TxGNN ที่ได้คะแนนสูงมีแนวโน้มสะท้อน **ความใกล้ชิดในเชิงโครงสร้างกราฟความรู้** ระหว่าง Clindamycin กับกลุ่มโรคกระจกตา มากกว่าจะบ่งชี้ความเชื่อมโยงทางคลินิกที่แท้จริง

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
การคาดการณ์ของ TxGNN อิงเฉพาะโมเดล (L5) โดยไม่มีการทดลองทางคลินิกหรือวรรณกรรมสนับสนุนเลย และกลไกการออกฤทธิ์ต้านแบคทีเรียของ Clindamycin ไม่สอดคล้องกับสาเหตุหลักของกระจกตาอักเสบแบบจุดซึ่งส่วนใหญ่เป็นไวรัสหรือสาเหตุที่ไม่ใช่แบคทีเรีย

**หากต้องการดำเนินการต่อต้อง:**
- ดึงข้อมูลกลไกการออกฤทธิ์ (MOA) เพิ่มเติมจาก DrugBank API (DB01190)
- ดาวน์โหลดและวิเคราะห์เอกสารกำกับยา (Package Insert) จากเว็บไซต์ TFDA เพื่อเติมข้อมูลคำเตือนและข้อห้ามใช้
- พิจารณาข้อบ่งใช้อันดับ 2 (**Exposure Keratitis**, L4) ซึ่งมีหลักฐานวรรณกรรม 4 ฉบับและมีเหตุผลทางกลไกที่แข็งแกร่งกว่า — อาจเหมาะสมกว่าสำหรับการศึกษาเชิงลึกในลำดับถัดไป
- ตรวจสอบสถานะการขึ้นทะเบียนในไทยซ้ำอีกครั้ง เนื่องจาก Clindamycin เป็นยาปฏิชีวนะที่ใช้กว้างขวางทั่วโลก
