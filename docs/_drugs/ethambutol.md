# Ethambutol: จากวัณโรคสู่การอักเสบของฝาปิดกล่องเสียง (Epiglottitis)

## สรุปสั้นๆ

Ethambutol เป็นยาต้านวัณโรคที่ใช้เป็นหนึ่งในสี่ยาหลักของสูตร HRZE มาตรฐาน ออกฤทธิ์ยับยั้งการสร้างผนังเซลล์ของเชื้อมัยโคแบคทีเรีย
โมเดล TxGNN คาดการณ์ว่าอาจมีบทบาทต่อ **การอักเสบของฝาปิดกล่องเสียง (Epiglottitis)** โดยอาศัยความเชื่อมโยงระหว่างวัณโรคกล่องเสียงที่อาจครอบคลุม epiglottis
ปัจจุบันมี **วรรณกรรม 2 ฉบับ** สนับสนุนแนวทางนี้ และยังไม่มีการทดลองทางคลินิกที่ลงทะเบียนไว้

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียดจาก DrugBank อย่างไรก็ตาม จากที่ทราบ Ethambutol ออกฤทธิ์โดยยับยั้งเอนไซม์อาราบิโนซิลทรานสเฟอเรส (arabinosyl transferase) ซึ่งจำเป็นต่อการสังเคราะห์อาราบิโนกาแลคแทน (arabinogalactan) ในผนังเซลล์ของเชื้อมัยโคแบคทีเรีย กลไกนี้มีความจำเพาะสูงต่อเชื้อ Mycobacterium และเป็นพื้นฐานของประสิทธิผลในการรักษาวัณโรคที่พิสูจน์แล้ว

ความเชื่อมโยงระหว่างวัณโรคกับ epiglottitis เกิดจากข้อเท็จจริงที่ว่าเชื้อ Mycobacterium tuberculosis สามารถแพร่กระจายไปยังอวัยวะนอกปอดได้ รวมถึงบริเวณกล่องเสียงซึ่งครอบคลุม epiglottis โดยมีรายงาน case series ว่า epiglottis เป็นหนึ่งในตำแหน่งที่พบการติดเชื้อวัณโรคกล่องเสียง (laryngeal tuberculosis)

อย่างไรก็ตาม ข้อมูลที่มีอยู่ครอบคลุมวัณโรคกล่องเสียงในภาพรวม ไม่มีหลักฐานที่ตรวจสอบ ethambutol ใน epiglottitis โดยตรง TxGNN น่าจะทำนายจากความใกล้ชิดของโหนดโรคในเครือข่ายความรู้ระหว่างการติดเชื้อทางเดินหายใจส่วนบนและวัณโรค

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
หลักฐานที่มีอยู่อยู่ในระดับ L4 (case series ทั้งหมด) โดยไม่มีการทดลองทางคลินิกที่ลงทะเบียนสำหรับ epiglottitis โดยตรง ประกอบกับ ethambutol ยังไม่ได้รับการจดทะเบียนในไทย และข้อมูลความปลอดภัย/MOA ยังขาดหาย ทำให้ไม่สามารถผ่านการประเมินเบื้องต้น (S1) ได้ในขณะนี้

**หากต้องการดำเนินการต่อต้อง:**
- ดาวน์โหลดและวิเคราะห์ MOA โดยละเอียดจาก DrugBank API (DB00330)
- ดาวน์โหลดหนังสือกำกับยาจากเว็บไซต์ TFDA เพื่อประเมินคำเตือนและข้อห้ามใช้
- ยืนยันว่า epiglottitis ที่เป็นเป้าหมายนั้นเป็นชนิดที่เกิดจากวัณโรค (tuberculous epiglottitis) ซึ่งจะทำให้กลไกของ ethambutol เกี่ยวข้องโดยตรง
- พิจารณายื่นขอจดทะเบียนยาในประเทศไทยก่อนดำเนินการศึกษาทางคลินิก หรือระบุแนวทางการเข้าถึงยาสำหรับการวิจัย
