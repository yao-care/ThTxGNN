# Lidocaine: จากยาชาเฉพาะที่สู่กระจกตาและเยื่อบุตาอักเสบแบบจุดชั้นเยื่อบุผิว

## สรุปสั้นๆ

Lidocaine เป็นยาชาเฉพาะที่และยารักษาภาวะหัวใจเต้นผิดจังหวะที่ใช้กันแพร่หลาย ออกฤทธิ์โดยบล็อกช่องโซเดียม (voltage-gated sodium channels) ของเซลล์ประสาทและกล้ามเนื้อหัวใจ
โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **กระจกตาและเยื่อบุตาอักเสบแบบจุดชั้นเยื่อบุผิว (Punctate Epithelial Keratoconjunctivitis)**
ปัจจุบัน**ไม่มีการทดลองทางคลินิกหรือวรรณกรรมสนับสนุนโดยตรง** หลักฐานอยู่ในระดับ L5 ซึ่งเป็นการทำนายจากโมเดลเท่านั้น

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียดจาก DrugBank (DG002) ตามที่ทราบ Lidocaine เป็นยาในกลุ่ม aminoamide local anesthetic ที่บล็อก voltage-gated sodium channels ป้องกันการเกิด action potential ในเส้นประสาทรับความรู้สึก จึงยับยั้งการส่งสัญญาณปวดและระคายเคืองได้อย่างมีประสิทธิภาพ นอกจากนี้ยังมีฤทธิ์ต้านการอักเสบในระดับที่พิสูจน์แล้วหลายการศึกษา

กระจกตาและเยื่อบุตาอักเสบแบบจุดชั้นเยื่อบุผิว (PEK) มีอาการเจ็บแสบและระคายเคืองรุนแรงจากความเสียหายของเซลล์เยื่อบุผิวกระจกตาเป็นบริเวณเล็กๆ กลไกการบล็อกเส้นประสาทรับความรู้สึกของ Lidocaine อาจช่วยบรรเทาอาการในระยะเฉียบพลันได้ผ่านเส้นทาง nociceptive pathway ของกระจกตา

อย่างไรก็ตาม มีข้อกังวลสำคัญด้านความปลอดภัย: การใช้ยาชาเฉพาะที่ในตาซ้ำๆ หรือระยะยาวอาจยับยั้งการงอกใหม่ของเส้นประสาทกระจกตา (corneal nerve regeneration) และชะลอการซ่อมแซมเยื่อบุผิว ซึ่งขัดแย้งโดยตรงกับเป้าหมายการรักษา PEK ที่ต้องการฟื้นฟูเยื่อบุผิว การพิจารณาต่อจึงต้องอาศัยหลักฐานทางคลินิกและก่อนคลินิกเพิ่มเติม

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
หลักฐานทั้งหมดอยู่ในระดับ L5 ซึ่งมีเพียงการทำนายจากโมเดล TxGNN โดยไม่มีการทดลองทางคลินิกหรือวรรณกรรมใดรองรับการใช้ Lidocaine รักษา Punctate Epithelial Keratoconjunctivitis โดยตรง และมีข้อกังวลเชิงกลไกว่ายาชาเฉพาะที่อาจขัดขวางกระบวนการซ่อมแซมกระจกตาแทนที่จะส่งเสริม

**หากต้องการดำเนินการต่อต้อง:**
- ดาวน์โหลดและวิเคราะห์ MOA จาก DrugBank API เพื่อยืนยันกลไกการออกฤทธิ์โดยละเอียด (DG002)
- ดาวน์โหลดเอกสารกำกับยา (PIL/SmPC) จาก Thai FDA เพื่อตรวจสอบคำเตือนและข้อห้ามใช้ (DG001)
- ค้นหาการศึกษาก่อนคลินิกที่ประเมินผลของ lidocaine ต่อ corneal epithelial healing และ nerve regeneration
- ประเมินความเป็นไปได้ของรูปแบบยา (เจล/สเปรย์/หยอดตา) ที่เหมาะกับการรักษา PEK
- พิจารณา indication ในอันดับที่ 6 (conjunctival disorder, L3) ซึ่งมีหลักฐานทางคลินิกมากกว่าและน่าจะเป็นเป้าหมายที่สมเหตุสมผลกว่าในการศึกษาต่อ
