# Alprazolam: จากโรควิตกกังวลสู่โรคนอนไม่หลับ

## สรุปสั้นๆ

Alprazolam เป็นยากลุ่มเบนโซไดอะซีพีน (benzodiazepine) ที่ใช้กันอย่างแพร่หลายทั่วโลกในการรักษาโรควิตกกังวลและโรคตื่นตระหนก แต่ปัจจุบันไม่มีทะเบียนยาในประเทศไทย
โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **โรคนอนไม่หลับ (Insomnia)**
ปัจจุบันมี **การทดลองทางคลินิก 7 รายการ** ที่เกี่ยวข้องทางอ้อม แต่ **ยังไม่มีวรรณกรรม** สนับสนุนโดยตรง

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

Alprazolam ออกฤทธิ์โดยการเพิ่มประสิทธิภาพการทำงานของตัวรับ GABA-A (positive allosteric modulator) ในระบบประสาทส่วนกลาง ส่งผลให้เกิดฤทธิ์สงบประสาทและช่วยให้หลับ (sedative-hypnotic effect) สามารถลดระยะเวลาที่ใช้ในการเข้าสู่การนอนหลับ (sleep onset latency) ได้

โรควิตกกังวลและโรคนอนไม่หลับมีความเชื่อมโยงกันทางคลินิกอย่างมาก ผู้ป่วยที่มีอาการวิตกกังวลมักประสบปัญหาการนอนไม่หลับร่วมด้วย และยากลุ่ม benzodiazepine หลายชนิด เช่น triazolam และ temazepam ได้รับการอนุมัติเป็นยานอนหลับแล้วในหลายประเทศ ดังนั้นการที่ alprazolam ซึ่งอยู่ในกลุ่มเดียวกันอาจมีผลต่อโรคนอนไม่หลับจึงสมเหตุสมผลในเชิงเภสัชวิทยา

อย่างไรก็ตาม alprazolam มีข้อจำกัดสำคัญ คือเปลี่ยนแปลงโครงสร้างการนอนหลับ โดยลดสัดส่วนของ slow-wave sleep และ REM sleep ทำให้คุณภาพการนอนโดยรวมลดลง นอกจากนี้ยังมีความเสี่ยงสูงต่อการเกิดการดื้อยา (tolerance) และการติดยา (dependence) จึง **ไม่ใช่ยาตัวเลือกแรก** สำหรับการรักษาโรคนอนไม่หลับตามแนวปฏิบัติทางคลินิกปัจจุบัน

## ข้อมูลการวางจำหน่ายในประเทศไทย

Alprazolam ไม่มีทะเบียนยาในประเทศไทย (0 ใบอนุญาต) ตามข้อมูลจากสำนักงานคณะกรรมการอาหารและยา (Thai FDA) ณ วันที่สืบค้น (24 มีนาคม 2569)

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
แม้ว่า alprazolam จะมีฤทธิ์สงบประสาทที่อาจช่วยเรื่องการนอนหลับได้ในทางทฤษฎี แต่หลักฐานจากการทดลองทางคลินิกที่มีอยู่ล้วนเกี่ยวข้องทางอ้อมเท่านั้น ไม่มีการศึกษาใดที่ทดสอบ alprazolam สำหรับโรคนอนไม่หลับโดยเฉพาะ นอกจากนี้ยายังไม่ได้จดทะเบียนในประเทศไทย และมีข้อกังวลสำคัญเรื่องการเปลี่ยนแปลงโครงสร้างการนอน การดื้อยา และการติดยา

**หากต้องการดำเนินการต่อต้อง:**
- เติมเต็มข้อมูลกลไกการออกฤทธิ์โดยละเอียด (MOA) จาก DrugBank API
- จัดหาคำเตือน ข้อห้ามใช้ และข้อมูลปฏิกิริยาระหว่างยาจากเอกสารกำกับยา
- ดำเนินการหรือระบุการทดลองทางคลินิกที่ศึกษา alprazolam สำหรับโรคนอนไม่หลับโดยเฉพาะ
- ประเมินเปรียบเทียบกับยานอนหลับที่มีหลักฐานแข็งแกร่งกว่า เช่น zolpidem, eszopiclone หรือ suvorexant
- พิจารณาสถานะการขึ้นทะเบียนยาในประเทศไทยก่อนจึงจะสามารถนำไปใช้ทางคลินิกได้

---

> **ข้อบ่งใช้ที่น่าสนใจอื่นๆ จาก TxGNN:**
> การทำนายอันดับ 3 **โรคกลัวที่โล่ง (Agoraphobia)** และอันดับ 10 **โรคกลัว (Phobic Disorder)** มีระดับหลักฐาน **L1** (มี ≥2 RCT หลายศูนย์สนับสนุน) พร้อมคำแนะนำ **Proceed with Guardrails** ซึ่งสอดคล้องกับข้อบ่งใช้ที่ได้รับการรับรองของ alprazolam ในต่างประเทศอยู่แล้ว และอาจเป็นเป้าหมายที่มีศักยภาพมากกว่าสำหรับการพิจารณาในบริบทของประเทศไทย

---

*ผลการวิเคราะห์นี้เป็นเพียงข้อมูลเพื่อการวิจัยเท่านั้น ไม่ถือเป็นคำแนะนำทางการแพทย์ การนำยาเก่ามาใช้ใหม่ต้องผ่านการทดลองทางคลินิกและได้รับอนุมัติจากหน่วยงานกำกับดูแลก่อนนำไปใช้จริง*
