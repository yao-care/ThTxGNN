# Ivermectin: จากยาต้านพยาธิสู่การติดเชื้อราแคนดิดาช่องคลอด

## สรุปสั้นๆ

Ivermectin เป็นยาต้านพยาธิที่มีประสิทธิผลพิสูจน์แล้วในการรักษาโรคพยาธิปรสิต เช่น พยาธิตาบอด (Onchocerciasis) พยาธิ Strongyloides และโรคเรื้อน (Scabies) โดยออกฤทธิ์ผ่านการเปิดช่องไอออน GluCl ในเซลล์ประสาทของปรสิต
โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **การติดเชื้อราแคนดิดาช่องคลอด (Vulvovaginal Candidiasis)**
อย่างไรก็ตาม ปัจจุบัน **ไม่มีการทดลองทางคลินิกและวรรณกรรมใดสนับสนุน** แนวทางนี้เลย และการคาดการณ์นี้ขาดพื้นฐานทางชีววิทยาที่ชัดเจน

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

Ivermectin ออกฤทธิ์โดยหลักผ่านการเปิดช่องไอออนคลอไรด์ที่ควบคุมด้วยกลูตาเมต (Glutamate-gated Chloride Channels, GluCl) ซึ่งพบเฉพาะในเซลล์ประสาทและกล้ามเนื้อของสัตว์ไม่มีกระดูกสันหลัง ส่งผลให้ปรสิตเป็นอัมพาตและตาย นอกจากนี้ยังมีรายงานว่า Ivermectin มีศักยภาพต้านไวรัสบางชนิดผ่านการยับยั้ง importin α/β pathway ที่ควบคุมการนำโปรตีนเข้านิวเคลียสของเซลล์ แม้หลักฐานทางคลินิกในส่วนนี้ยังจำกัดมาก

**การคาดการณ์นี้ขาดพื้นฐานทางชีววิทยาอย่างมีนัยสำคัญ** เชื้อรา Candida เป็น eukaryote ที่ไม่มี GluCl channels อันเป็นเป้าหมายหลักของ Ivermectin และไม่มีกลไกใดที่ทราบของยานี้ที่สามารถยับยั้งเชื้อราได้ ข้อสังเกตที่สำคัญคือทั้ง 9 ข้อบ่งใช้ที่โมเดลทำนายล้วนอยู่ในกลุ่มโรคแคนดิดาและโรคบริเวณอวัยวะสืบพันธุ์ทั้งสิ้น ซึ่งบ่งชี้ว่าการคาดการณ์เหล่านี้น่าจะมาจากความสัมพันธ์เชิงโครงสร้างกราฟ (graph topology artifact) ใน TxGNN ระหว่าง node โรคติดเชื้อบริเวณอวัยวะสืบพันธุ์ มากกว่าความเชื่อมโยงทางชีวเคมีที่แท้จริง

ปัจจุบันไม่มีการศึกษาก่อนคลินิก (in vitro/in vivo) ใดที่แสดงฤทธิ์ต้านเชื้อรา Candida ของ Ivermectin และไม่มีวรรณกรรมหรือการทดลองทางคลินิกใดสนับสนุนข้อบ่งใช้นี้ ทำให้ระดับหลักฐานอยู่ที่ L5 เท่านั้น

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
แม้คะแนน TxGNN จะสูงถึง 99.95% แต่การคาดการณ์นี้ขาดพื้นฐานทางชีววิทยาอย่างสมบูรณ์ Ivermectin ไม่มีฤทธิ์ต้านเชื้อราที่ทราบ ไม่มีการทดลองทางคลินิกหรือวรรณกรรมรองรับ และยังไม่มีทะเบียนยาในประเทศไทย ระดับหลักฐาน L5 ไม่เพียงพอสำหรับการดำเนินการต่อ

**หากต้องการดำเนินการต่อต้อง:**
- ดาวน์โหลดเอกสารกำกับยาจาก TFDA เพื่อประเมินข้อห้ามใช้และคำเตือนหลัก
- สืบค้น DrugBank API เพื่อยืนยันกลไกการออกฤทธิ์ที่สมบูรณ์
- ทบทวนวรรณกรรมก่อนคลินิก (in vitro) เพื่อตรวจสอบว่ามีหลักฐานฤทธิ์ต้านเชื้อรา Candida ของ Ivermectin หรือไม่ก่อนพิจารณาขั้นตอนถัดไปใดๆ
