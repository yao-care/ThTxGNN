# Erythromycin: จากการรักษาการติดเชื้อแบคทีเรียสู่กระจกตาอักเสบชนิดจุด (Punctate Epithelial Keratoconjunctivitis)

## สรุปสั้นๆ

Erythromycin เป็นยาปฏิชีวนะกลุ่ม macrolide ที่ใช้รักษาการติดเชื้อแบคทีเรียหลากหลายชนิด ทั้งทางเดินหายใจ ผิวหนัง และโรคติดต่อทางเพศสัมพันธ์ รวมถึงมีฤทธิ์ต้านเชื้อก่อโรคภายในเซลล์อย่าง Chlamydia trachomatis โมเดล TxGNN คาดการณ์ว่ายานี้อาจมีผลต่อ **กระจกตาอักเสบชนิดจุด (Punctate Epithelial Keratoconjunctivitis)** ด้วยคะแนนการทำนายสูงถึง 99.89% ปัจจุบันมี **วรรณกรรม 2 ฉบับ** ที่เกี่ยวข้องสนับสนุนแนวทางนี้ในเชิงทางอ้อม แต่ยังไม่มีการทดลองทางคลินิกโดยตรงสำหรับข้อบ่งใช้นี้

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียดจากฐานข้อมูล DrugBank (Data Gap) อย่างไรก็ตาม จากหลักฐานที่ปรากฏในวรรณกรรมภายใน Evidence Pack นี้ Erythromycin เป็นยาปฏิชีวนะกลุ่ม macrolide ที่ออกฤทธิ์ยับยั้งการสังเคราะห์โปรตีนของแบคทีเรียโดยจับกับ ribosomal subunit 50S มีประสิทธิผลต่อเชื้อก่อโรคภายในเซลล์ (intracellular pathogens) โดยเฉพาะ Chlamydia trachomatis ซึ่งเป็นสาเหตุสำคัญของโรคตาอักเสบหลายชนิด รวมถึงยังมีคุณสมบัติต้านการอักเสบในระดับท้องถิ่นที่อาจช่วยบรรเทาความเสียหายของเยื่อบุผิวกระจกตาได้

กระจกตาอักเสบชนิดจุด (Punctate Epithelial Keratoconjunctivitis) เป็นภาวะอักเสบของพื้นผิวกระจกตาที่อาจเกิดจากเชื้อแบคทีเรีย ไวรัส หรือปัจจัยอื่น ความสามารถของ Erythromycin ในการซึมผ่านเข้าสู่เนื้อเยื่อตาและยับยั้งเชื้อก่อโรคภายในเซลล์ทำให้เกิดความเป็นไปได้ทางทฤษฎีสำหรับการนำมาใช้ในข้อบ่งใช้นี้ ทั้งนี้ Erythromycin ขนาดยาสำหรับตา (ophthalmic ointment) เป็นรูปแบบที่ใช้ในทางคลินิกเพื่อรักษาการติดเชื้อตาอยู่แล้ว

อย่างไรก็ตาม หลักฐานทั้ง 2 ฉบับที่มีอยู่ในปัจจุบันเป็นเพียงหลักฐานทางอ้อม ไม่มีการศึกษาทางคลินิกที่ทดสอบ Erythromycin โดยตรงในข้อบ่งใช้ Punctate Epithelial Keratoconjunctivitis การคาดการณ์นี้มาจากความสัมพันธ์ในกราฟความรู้ (knowledge graph) ของ TxGNN ซึ่งยังต้องการการตรวจสอบเพิ่มเติมในเชิงคลินิก

## ข้อพิจารณาด้านความปลอดภัย

กรุณาดูข้อมูลความปลอดภัยในเอกสารกำกับยา

> **หมายเหตุ:** ข้อมูลคำเตือนหลัก ข้อห้ามใช้ และปฏิกิริยาระหว่างยาจาก TFDA/DrugBank ยังอยู่ในขั้น Data Gap กรุณาดาวน์โหลดเอกสารกำกับยา (Package Insert) จากเว็บไซต์ TFDA และสืบค้น DrugBank API เพิ่มเติมก่อนนำไปประเมินความปลอดภัยในระดับถัดไป

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
หลักฐานอยู่ในระดับ L4 (การศึกษาก่อนคลินิก/หลักฐานทางกลไก) โดยไม่มีการทดลองทางคลินิกที่ตรวจสอบ Erythromycin โดยตรงสำหรับ Punctate Epithelial Keratoconjunctivitis วรรณกรรม 2 ฉบับที่มีอยู่เป็นหลักฐานทางอ้อมระดับ Tier 3 ทั้งสิ้น ยังไม่เพียงพอสำหรับการประเมินเพื่อนำไปใช้ทางคลินิก นอกจากนี้ ข้อมูลความปลอดภัย (TFDA warnings, DDI, contraindications) และกลไกการออกฤทธิ์ยังเป็น Data Gap ซึ่งถือเป็น Blocking issue สำหรับการประเมิน S1

**หากต้องการดำเนินการต่อต้อง:**
- ดาวน์โหลดและวิเคราะห์เอกสารกำกับยา (PDF) จากเว็บไซต์ TFDA เพื่อให้ได้ข้อมูลคำเตือนและข้อห้ามใช้ (แก้ไข DG001)
- สืบค้นกลไกการออกฤทธิ์ (MOA) จาก DrugBank API (แก้ไข DG002)
- สืบค้นการทดลองทางคลินิกเพิ่มเติมด้วยคำค้น "erythromycin ophthalmic" หรือ "macrolide keratoconjunctivitis" เพื่อเพิ่มฐานหลักฐาน
- พิจารณาศึกษาก่อนคลินิกในแบบจำลองโรคตาเพื่อยืนยันกลไกต้านเชื้อในระดับพื้นผิวกระจกตาก่อนยกระดับเป็น S2
