# Gliclazide: จากเบาหวานชนิดที่ 2 สู่ Classic Stiff Person Syndrome

## สรุปสั้นๆ

Gliclazide เป็นยาในกลุ่ม Sulfonylurea ที่ใช้รักษาโรคเบาหวานชนิดที่ 2 โดยกระตุ้นการหลั่งอินซูลินจากเซลล์ β ของตับอ่อนผ่านการปิดช่อง K-ATP โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **Classic Stiff Person Syndrome** ซึ่งเป็นโรคระบบประสาทภูมิคุ้มกันที่หายากมาก โดยการทำนายนี้มีแนวทางกลไกที่น่าสนใจจากการเชื่อมโยงผ่าน GAD65 แต่ปัจจุบัน **ไม่มีการทดลองทางคลินิกหรือวรรณกรรมใด** ที่สนับสนุนโดยตรง หลักฐานทั้งหมดอยู่ในระดับการทำนายจากโมเดลเท่านั้น

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียดจากฐานข้อมูล ตามข้อมูลที่ทราบ Gliclazide ยับยั้งช่อง K-ATP (SUR1/Kir6.2) บนเซลล์ β ของตับอ่อน ทำให้เซลล์ดีโพลาไรซ์และปล่อยอินซูลิน นอกจากนี้ยังมีคุณสมบัติพิเศษ ได้แก่ การกำจัดอนุมูลอิสระจากโครงสร้างไนโตรเจนไบไซคลิก (azabicyclo-octyl) และฤทธิ์ปกป้องหลอดเลือดขนาดเล็กผ่านการปรับการแข็งตัวของเลือด

ความเชื่อมโยงที่ทำให้ TxGNN ให้คะแนนสูงคือ Classic Stiff Person Syndrome มีแอนติเจนหลักคือ **GAD65 (Glutamic Acid Decarboxylase 65)** ซึ่งเป็นเอนไซม์ที่แสดงออกในเซลล์ β ของตับอ่อนเช่นเดียวกัน ผู้ป่วย Stiff Person Syndrome มักพบแอนติบอดีต่อ GAD65 ในระดับสูงมาก และในผู้ป่วยเบาหวานชนิดที่ 1 ก็พบแอนติบอดีชนิดเดียวกัน บ่งชี้ว่าทั้งสองโรคมีรากเหง้าทางภูมิคุ้มกันร่วมกัน โมเดล TxGNN อาจตรวจพบความใกล้เคียงเชิงโทโพโลยีนี้ในกราฟความรู้และแปลงเป็นคะแนนการทำนายที่สูง

อย่างไรก็ตาม **ยังไม่มีหลักฐานว่า Gliclazide สามารถปรับเปลี่ยนการตอบสนองภูมิคุ้มกันต่อ GAD65 ได้โดยตรง** การทำนายนี้เป็นเพียงสมมติฐานจากความสัมพันธ์เชิงโครงข่ายในกราฟความรู้ ยังต้องการการศึกษาก่อนคลินิกเพื่อพิสูจน์ก่อนจะพิจารณาขยับสู่ขั้นตอนใดๆ เพิ่มเติม

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
หลักฐานทั้งหมดอยู่ในระดับ L5 คือการทำนายจากโมเดลเท่านั้น ไม่มีการทดลองทางคลินิก วรรณกรรม หรือแม้แต่การศึกษาก่อนคลินิกที่สนับสนุนการใช้ Gliclazide ใน Classic Stiff Person Syndrome และยังไม่ได้ขึ้นทะเบียนในประเทศไทย

**หากต้องการดำเนินการต่อต้อง:**
- สืบค้นและยืนยันกลไกการออกฤทธิ์ (MOA) โดยละเอียดจาก DrugBank API และดาวน์โหลดเอกสารกำกับยา (TFDA/EMA) เพื่อประเมินความปลอดภัยขั้นต้น
- ดำเนินการศึกษาก่อนคลินิก (in vitro) เพื่อทดสอบผลของ Gliclazide ต่อการตอบสนองภูมิคุ้มกัน GAD65 เช่น การทดสอบในเซลล์เพาะเลี้ยงจากผู้ป่วย Stiff Person Syndrome
- หาก in vitro มีสัญญาณน่าสนใจ ให้ยกระดับเป็นการศึกษาในแบบจำลองสัตว์ก่อนพิจารณาการทดลองในมนุษย์
- ตรวจสอบสถานะการขึ้นทะเบียนยา Gliclazide ในประเทศไทยหากต้องการดำเนินการทางคลินิกในอนาคต
