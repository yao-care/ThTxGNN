# Doxycycline: จากยาปฏิชีวนะกว้างสเปคตรัมสู่กระจกตาเยื่อบุตาอักเสบแบบจุด

## สรุปสั้นๆ

Doxycycline เป็นยาปฏิชีวนะกลุ่มเตตราไซคลินที่ออกฤทธิ์ต้านเชื้อแบคทีเรียในวงกว้าง ใช้รักษาการติดเชื้อคลาไมเดีย โรคไลม์ โรคสครับไทฟัส และการติดเชื้อริกเก็ตเซีย
โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **กระจกตาเยื่อบุตาอักเสบแบบจุดที่เยื่อบุผิว (Punctate Epithelial Keratoconjunctivitis)**
ปัจจุบันมี **การทดลองทางคลินิก 0 รายการ** และ **วรรณกรรม 1 ฉบับ** (รายงานผู้ป่วย ปี 1992) สนับสนุนแนวทางนี้

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียดจากฐานข้อมูล DrugBank ตามที่ทราบโดยทั่วไป Doxycycline เป็นยาปฏิชีวนะกลุ่มเตตราไซคลินซึ่งออกฤทธิ์ยับยั้งการสังเคราะห์โปรตีนของแบคทีเรียโดยการจับกับหน่วยย่อย 30S ของไรโบโซม ครอบคลุมทั้งแบคทีเรียกรัมบวก กรัมลบ และเชื้อในเซลล์ (intracellular) โดยเฉพาะ *Chlamydia trachomatis* และ Rickettsia ซึ่งเป็นสาเหตุสำคัญของโรคที่ตา

หลักฐานสำคัญจากรายงานผู้ป่วยปี 1992 ชี้ให้เห็นว่า Doxycycline สามารถรักษาการติดเชื้อ *C. trachomatis* ที่เยื่อบุตาได้ (follicular conjunctivitis) แต่หลังจากการอักเสบสงบ ผู้ป่วยบางรายยังคงมีรอยโรคแบบจุดที่กระจกตาต่อเนื่อง (Punctate Epithelial Keratitis) ซึ่งน่าจะเกิดจากความเสียหายของเยื่อบุกระจกตาที่เหลือค้างหลังการติดเชื้อ

กระบวนการนี้สมเหตุสมผลในเชิงกลไก เนื่องจาก Doxycycline รักษาเชื้อต้นเหตุได้ แต่การป้องกันหรือรักษา PEK ที่เกิดขึ้นแล้วโดยตรงยังเป็นคำถามที่ยังไม่ได้รับการตอบด้วยการศึกษาเชิงคาดการณ์ จำเป็นต้องมีงานวิจัยเพิ่มเติมเพื่อยืนยันสมมติฐานนี้

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Research Question**

**เหตุผล:**
หลักฐานปัจจุบันอยู่ในระดับ L4 มีเพียงรายงานผู้ป่วยเชิงสังเกต 1 ฉบับจากปี 1992 ซึ่งไม่ได้ออกแบบมาเพื่อพิสูจน์ประสิทธิผลของ Doxycycline ใน PEK โดยตรง และยังไม่พบทะเบียนยาในฐานข้อมูล Thai FDA ทำให้ยังไม่เพียงพอต่อการประเมินความเป็นไปได้ในเชิงคลินิก

**หากต้องการดำเนินการต่อต้อง:**
- ดึงข้อมูลกลไกการออกฤทธิ์โดยละเอียดจาก DrugBank API (แก้ไข DG002)
- ดาวน์โหลดและวิเคราะห์หนังสือกำกับยาจาก Thai FDA เพื่อประเมินคำเตือนและข้อห้ามใช้ (แก้ไข DG001)
- ออกแบบ Pilot study เชิงคาดการณ์เพื่อประเมินบทบาทของ Doxycycline ในการป้องกัน PEK ภายหลังการติดเชื้อคลาไมเดีย
- ตรวจสอบสถานะการขึ้นทะเบียน Doxycycline กับ Thai FDA โดยตรง เนื่องจากฐานข้อมูลอาจไม่ครบถ้วน
