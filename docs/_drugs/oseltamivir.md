# Oseltamivir: จากไข้หวัดใหญ่สู่ไตอักเสบจากเชื้อแบคทีเรีย (Pyelonephritis)

## สรุปสั้นๆ

Oseltamivir (Tamiflu) เป็นยาต้านไวรัสที่ใช้รักษาและป้องกันโรคไข้หวัดใหญ่ชนิด A และ B โดยออกฤทธิ์ยับยั้งเอนไซม์ neuraminidase ของไวรัส ทำให้ไวรัสใหม่ไม่สามารถแตกตัวออกจากเซลล์และแพร่กระจายได้
โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **ไตอักเสบจากเชื้อแบคทีเรีย (Pyelonephritis)** ด้วยคะแนนสูงถึง 97.85%
อย่างไรก็ตาม ปัจจุบัน**ไม่มีการทดลองทางคลินิก** และมี**วรรณกรรมเพียง 1 ฉบับ**ที่เกี่ยวข้อง ซึ่งไม่ได้สนับสนุนการใช้ยาในข้อบ่งใช้นี้แต่อย่างใด

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียดจากฐานข้อมูล DrugBank ตามข้อมูลที่ทราบ Oseltamivir เป็นสารยับยั้งเอนไซม์ neuraminidase (NA) ของไวรัสไข้หวัดใหญ่แบบ competitive inhibition โดยจับกับ active site ของ NA ทำให้ไวรัสไม่สามารถแยกตัวออกจากเซลล์ที่ติดเชื้อและแพร่ไปยังเซลล์ใหม่ได้

**ความเชื่อมโยงกับไตอักเสบจากเชื้อแบคทีเรียนั้นแทบไม่มีทางชีววิทยาใดรองรับ** ไตอักเสบ (Pyelonephritis) เกิดจากเชื้อแบคทีเรียเป็นหลัก (ส่วนใหญ่คือ *E. coli*) ที่ลุกลามจากกระเพาะปัสสาวะขึ้นสู่ไต ซึ่งเป็นกลไกที่ไม่มีจุดตัดทางชีวเคมีกับการยับยั้ง viral neuraminidase วรรณกรรมเดียวที่พบ (PMID 26033556) กล่าวถึงไข้หวัดใหญ่ในหญิงตั้งครรภ์ ไม่ใช่ไตอักเสบ

การคาดการณ์นี้น่าจะเกิดจาก **topology เชิงกราฟ** ภายใน knowledge graph ที่เชื่อมโยง Oseltamivir เข้ากับภาวะในหญิงตั้งครรภ์ ซึ่งเป็นกลุ่มที่มีความเสี่ยงต่อทั้งไข้หวัดใหญ่และการติดเชื้อทางเดินปัสสาวะ ส่งผลให้เกิดการเชื่อมทางอ้อมในกราฟโดยไม่มีนัยสำคัญทางคลินิก

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
การคาดการณ์นี้อยู่ในระดับหลักฐาน L5 (เฉพาะการทำนายจากโมเดล ไม่มีการศึกษาจริง) และขาดกลไกทางชีววิทยาที่เชื่อมโยง Oseltamivir กับการรักษาไตอักเสบจากเชื้อแบคทีเรีย ทำให้ไม่มีเหตุผลเพียงพอสำหรับการลงทุนทรัพยากรวิจัยในขณะนี้

**หากต้องการดำเนินการต่อต้อง:**
- ค้นหาหลักฐานก่อนคลินิกที่แสดงว่า Oseltamivir หรือการยับยั้ง neuraminidase ของโฮสต์ (NEU1) มีผลต่อเชื้อก่อโรคในทางเดินปัสสาวะ
- ดาวน์โหลดและวิเคราะห์เอกสารกำกับยา (Package Insert) จาก TFDA เพื่อประเมินความปลอดภัยเบื้องต้น
- ระบุกลไกการออกฤทธิ์ (MOA) โดยละเอียดจาก DrugBank API เพื่ออุดช่องว่างข้อมูล DG002
