# Naproxen: จากยาแก้ปวดและต้านการอักเสบสู่กลุ่มอาการนิ้วสั้น-นิ้วเชื่อม

## สรุปสั้นๆ

Naproxen เป็นยาต้านการอักเสบชนิดไม่ใช่สเตียรอยด์ (NSAID) ที่ออกฤทธิ์ยับยั้ง COX-1/COX-2 เพื่อลดการสังเคราะห์พรอสตาแกลนดิน ใช้บรรเทาอาการปวดและอักเสบมาอย่างยาวนานในระดับสากล
โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **กลุ่มอาการนิ้วสั้น-นิ้วเชื่อม (Brachydactyly-Syndactyly Syndrome)** ซึ่งเป็นโรคพัฒนาการกระดูกหายาก
ปัจจุบัน **ไม่มีการทดลองทางคลินิกหรือวรรณกรรมใดสนับสนุนแนวทางนี้เลย** หลักฐานอยู่ในระดับ L5 (การทำนายจากโมเดลเท่านั้น)

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียดจากฐานข้อมูลอย่างเป็นทางการ อย่างไรก็ตาม Naproxen เป็น NSAID ที่ยับยั้ง COX-1/COX-2 ส่งผลให้การสังเคราะห์พรอสตาแกลนดิน E2 (PGE2) ลดลง

กลุ่มอาการนิ้วสั้น-นิ้วเชื่อมมักมีต้นเหตุจากความผิดปกติของวิถีสัญญาณ BMP/SMAD ในช่วงพัฒนาการของนิ้วมือ-นิ้วเท้า ซึ่ง PGE2 มีบทบาทในการควบคุมการแบ่งตัวของเซลล์กระดูกอ่อนและการแสดงออกของตัวรับ BMP ในทางทฤษฎีจึงมีช่องทางที่ Naproxen อาจเปลี่ยนแปลงสมดุลแกน PGE2/BMP ได้

อย่างไรก็ตาม ความเชื่อมโยงนี้เป็นการอนุมานทางอ้อมในระดับสูงมาก และโรคนิ้วสั้น-นิ้วเชื่อมที่เกิดจากการกลายพันธุ์ทางพันธุกรรมมักมีความผิดปกติของโครงสร้างที่เกิดขึ้นถาวรตั้งแต่ช่วงพัฒนาการ ซึ่งการยับยั้ง COX-2 ไม่น่าจะแก้ไขได้ **ปัจจุบันไม่มีการศึกษาทางคลินิกหรือก่อนคลินิกใดๆ รองรับ** และคะแนน TxGNN ที่สูงน่าจะสะท้อนโครงสร้างความใกล้ชิดในกราฟความรู้มากกว่าหลักฐานทางเภสัชวิทยาที่แท้จริง

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
หลักฐานทั้งหมดอยู่ในระดับ L5 คือการทำนายจากโมเดลเท่านั้น ไม่มีการทดลองทางคลินิก วรรณกรรม หรือการศึกษาก่อนคลินิกใดๆ รองรับ และความเชื่อมโยงทางกลไกระหว่าง Naproxen กับโรคพัฒนาการกระดูกหายากนี้เป็นการอนุมานทางอ้อมที่ยังไม่ผ่านการพิสูจน์

**หากต้องการดำเนินการต่อต้อง:**
- ดึงข้อมูลกลไกการออกฤทธิ์ (MOA) จาก DrugBank API (DB00788) เพื่อยืนยันรายละเอียด
- ดาวน์โหลดและวิเคราะห์เอกสารกำกับยาจากหน่วยงานกำกับยาสากล (FDA/EMA) เพื่อประเมินความปลอดภัย
- ทบทวนหลักฐานก่อนคลินิกที่เกี่ยวข้องกับ COX-2/PGE2 ในโรคกระดูกพัฒนาการก่อนตัดสินใจเดินหน้า
- ตรวจสอบสถานะการขึ้นทะเบียนยาในประเทศไทยก่อนวางแผนการศึกษาใดๆ
