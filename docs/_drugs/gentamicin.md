# Gentamicin: จากยาปฏิชีวนะแกรมลบสู่โรคข้ออักเสบรูมาตอยด์

## สรุปสั้นๆ

Gentamicin เป็นยาปฏิชีวนะกลุ่ม Aminoglycoside ที่ใช้รักษาการติดเชื้อแบคทีเรียแกรมลบชนิดรุนแรง เช่น ปอดอักเสบ ภาวะโลหิตเป็นพิษ และการติดเชื้อในระบบทางเดินปัสสาวะ โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **โรคข้ออักเสบรูมาตอยด์ (Rheumatoid Arthritis)** ปัจจุบันมี **การทดลองทางคลินิก 1 รายการ** และ **วรรณกรรม 20 ฉบับ** ที่เชื่อมโยงกับแนวทางนี้ แต่หลักฐานส่วนใหญ่เป็นบริบทการติดเชื้อในผู้ป่วย RA หรือการป้องกันการติดเชื้อหลังผ่าตัด มิใช่การรักษา RA โดยตรง

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียดจากฐานข้อมูล DrugBank อย่างไรก็ตาม Gentamicin เป็นยาปฏิชีวนะที่ยับยั้งการสังเคราะห์โปรตีนของแบคทีเรียโดยจับกับซับยูนิต 30S ของไรโบโซม มีฤทธิ์ครอบคลุมแบคทีเรียแกรมลบหลายชนิด นอกจากนี้ Gentamicin ยังมีคุณสมบัติ **readthrough effect** ซึ่งสามารถทำให้ไรโบโซมของเซลล์สัตว์อ่านผ่านโคดอนหยุด (nonsense codon) ได้ — คุณสมบัตินี้กำลังอยู่ระหว่างการศึกษาสำหรับโรคทางพันธุกรรมบางชนิด

ความเชื่อมโยงระหว่าง Gentamicin กับโรคข้ออักเสบรูมาตอยด์มีพื้นฐานทางอ้อม 2 ประการ ได้แก่ (1) ฤทธิ์ต้านแบคทีเรียที่อาจช่วยลดการกระตุ้นข้ออักเสบที่เกิดจากการติดเชื้อ (reactive arthritis) และ (2) Gentamicin เป็นส่วนผสมหลักในกระดูกซีเมนต์สำหรับการผ่าตัดเปลี่ยนข้อในผู้ป่วย RA เพื่อป้องกันการติดเชื้อบริเวณรอยต่อ ซึ่งสะท้อนว่ายานี้ถูกใช้ในวงการรักษา RA อยู่แล้วในฐานะยาเสริม

อย่างไรก็ตาม Gentamicin ไม่มีกลไกที่เข้าไปยับยั้งกระบวนการภูมิคุ้มกันผิดปกติที่เป็นรากฐานของ RA เช่น การกระตุ้น T-cell หรือการอักเสบของเยื่อหุ้มข้อ (synovitis) ดังนั้นการคาดการณ์ของ TxGNN จึงน่าจะมาจากความสัมพันธ์ทางโครงข่ายในกราฟยา-โรค มากกว่าจะมีกลไกเชิงเหตุผลที่ชัดเจน

## ข้อพิจารณาด้านความปลอดภัย

กรุณาดูข้อมูลความปลอดภัยในเอกสารกำกับยา

> ⚠️ **หมายเหตุเพิ่มเติม:** วรรณกรรมในกลุ่มการทำนาย diabetic nephropathy (rank 8) ระบุชัดว่า Gentamicin เป็นยาที่มีพิษต่อไตที่ทราบกันดี และมักถูกใช้เป็นโมเดลในการเหนี่ยวนำไตบาดเจ็บในสัตว์ทดลอง ข้อมูลนี้ควรนำมาพิจารณาเป็นข้อกังวลหลักด้านความปลอดภัยก่อนการศึกษาใด ๆ ในมนุษย์

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
วรรณกรรมและการทดลองทางคลินิกที่มีอยู่ทั้งหมดไม่ได้สนับสนุนการใช้ Gentamicin เป็นยารักษา RA โดยตรง — หลักฐานส่วนใหญ่บรรยายบทบาทของยาในการป้องกันและรักษาการติดเชื้อในผู้ป่วย RA หรือในกระดูกซีเมนต์ผ่าตัดเปลี่ยนข้อ ซึ่งไม่เข้าข่ายการนำยาเก่ามาใช้ใหม่ อีกทั้งยังมีข้อกังวลสำคัญด้านพิษต่อไตและหูในการใช้ระยะยาว

**หากต้องการดำเนินการต่อต้อง:**
- ดาวน์โหลดและวิเคราะห์เอกสารกำกับยา (package insert) จากเว็บไซต์ อย. เพื่อรวบรวมคำเตือน ข้อห้ามใช้ และข้อควรระวังที่ครบถ้วน
- ตรวจสอบข้อมูล MOA จาก DrugBank API เพื่อยืนยันเส้นทางทางชีววิทยาที่อาจเกี่ยวข้องกับ RA
- ดำเนินการศึกษาก่อนคลินิก (in vitro) เพื่อทดสอบว่า Gentamicin มีผลต่อ inflammatory cytokines ที่จำเพาะกับ RA (เช่น TNF-α, IL-6, IL-1β) หรือไม่
- ประเมินโปรไฟล์ความปลอดภัยด้านพิษต่อไตและหูในบริบทการใช้ระยะยาว ก่อนพิจารณาออกแบบการศึกษาทางคลินิก
