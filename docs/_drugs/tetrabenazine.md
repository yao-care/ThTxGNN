<h3>為什麼這個預測合理？</h3>

<p>### 作用機轉支持</p>

<p>Tetrabenazine 透過可逆性抑制囊泡單胺轉運蛋白 2（VMAT2），減少多巴胺、血清素和正腎上腺素在神經末梢的儲存和釋放，從而減少過度運動症狀。</p>

### 主要不良反應

- **常見**：嗜睡、疲勞、帕金森症狀、噁心
- **較少見**：憂鬱、焦慮、失眠、靜坐不能
- **嚴重**：
  - **憂鬱與自殺風險**（黑框警告）
  - 惡性神經抑制症候群（NMS）
  - QT 延長

### 黑框警告

**憂鬱與自殺風險**：
- Tetrabenazine 可能導致或惡化憂鬱症
- 患者可能出現自殺意念
- 需密切監測情緒變化
- 有重度憂鬱症病史者禁用

### 特殊族群注意事項

- **肝功能不全**：需減量
- **CYP2D6 代謝差者**：需減量
- **老年人**：謹慎使用，副作用風險增加
- **懷孕**：Category C
