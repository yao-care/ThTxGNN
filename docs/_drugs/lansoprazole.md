# Lansoprazole: จากยาลดกรดสู่ภาวะน้ำดีไหลย้อนกลับเข้ากระเพาะ (Duodenogastric Reflux)

## สรุปสั้นๆ

Lansoprazole เป็นยากลุ่มยับยั้งโปรตอนปั๊ม (Proton Pump Inhibitor, PPI) ที่ใช้รักษาโรคที่เกี่ยวกับกรดในระบบทางเดินอาหาร เช่น แผลในกระเพาะอาหาร กรดไหลย้อน และการติดเชื้อ H. pylori
โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **ภาวะน้ำดีไหลย้อนกลับเข้ากระเพาะ (Duodenogastric Reflux)** ด้วยคะแนนสูงถึง 99.69%
ปัจจุบันมี **วรรณกรรม 2 ฉบับ** ที่เกี่ยวข้อง โดยยังไม่มีการทดลองทางคลินิกที่ลงทะเบียน และหลักฐานที่มีอยู่ชี้ให้เห็น **ข้อกังวลด้านความปลอดภัย** มากกว่าการสนับสนุนประสิทธิผล

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียดจากฐานข้อมูล DrugBank สำหรับ Evidence Pack นี้ อย่างไรก็ตาม จากหลักฐานในวรรณกรรมที่รวบรวมได้ Lansoprazole เป็นยา PPI ที่ออกฤทธิ์โดยยับยั้ง H+/K+-ATPase (โปรตอนปั๊ม) ในเซลล์ปาไรทัลของกระเพาะอาหารอย่างไม่ผันกลับ ส่งผลให้กรดในกระเพาะลดลงอย่างมีนัยสำคัญทั้งในภาวะปกติและเมื่อถูกกระตุ้น ยานี้ได้รับการพิสูจน์ประสิทธิผลในการรักษาแผลในกระเพาะอาหาร แผลในลำไส้เล็กส่วนต้น กรดไหลย้อน และ Zollinger-Ellison syndrome

ในทางทฤษฎี การลดความเป็นกรดในกระเพาะอาจช่วยบรรเทาความเสียหายของเยื่อบุกระเพาะที่เกิดจากน้ำดีและน้ำย่อยที่ไหลย้อนกลับจากลำไส้เล็ก ซึ่งเป็นลักษณะเฉพาะของภาวะ Duodenogastric Reflux กลไกนี้ฟังดูสอดคล้องกับการออกฤทธิ์หลักของ Lansoprazole

อย่างไรก็ตาม มีข้อค้นพบสำคัญที่ต้องพิจารณา: การศึกษาในสัตว์ทดลอง (PMID 15052437) รายงานว่า Lansoprazole อาจ **ส่งเสริม** การเกิดมะเร็งในต่อมของกระเพาะอาหารหนูที่มีภาวะน้ำดีไหลย้อน ซึ่งเป็นสัญญาณเตือนด้านความปลอดภัยที่ขัดแย้งกับวัตถุประสงค์การรักษา และทำให้หลักฐานที่มีอยู่สนับสนุนการระมัดระวังมากกว่าการพัฒนาต่อ

## ข้อมูลการวางจำหน่ายในประเทศไทย

Lansoprazole **ยังไม่ได้ขึ้นทะเบียนในประเทศไทย** ณ วันที่สืบค้น (24 มีนาคม 2569) ไม่พบรายการทะเบียนยาใดๆ ในฐานข้อมูล Thai FDA

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
หลักฐานสนับสนุนอยู่ในระดับ L4 (การศึกษาก่อนคลินิก) เท่านั้น และการศึกษาในสัตว์ระบุว่า Lansoprazole อาจเพิ่มความเสี่ยงการเกิดมะเร็งกระเพาะอาหารในบริบทของภาวะน้ำดีไหลย้อน ซึ่งเป็นข้อกังวลด้านความปลอดภัยที่ขัดแย้งกับเป้าหมายการรักษา และต้องได้รับการตรวจสอบก่อนพัฒนาต่อ

**หากต้องการดำเนินการต่อต้อง:**
- ทำการศึกษาก่อนคลินิกเพิ่มเติมเพื่อชี้แจงผลของ Lansoprazole ต่อความเสี่ยงมะเร็งในภาวะ Duodenogastric Reflux โดยเฉพาะ
- รวบรวมข้อมูลกลไกการออกฤทธิ์ (MOA) โดยละเอียดจาก DrugBank API (Data Gap DG002)
- ดาวน์โหลดและวิเคราะห์เอกสารกำกับยาเพื่อรับข้อมูลคำเตือนและข้อห้ามใช้ (Data Gap DG001)
- พิจารณาว่ายา PPI ชนิดอื่นที่มีทะเบียนในประเทศไทยอยู่แล้วอาจเป็นทางเลือกที่เหมาะสมกว่า
