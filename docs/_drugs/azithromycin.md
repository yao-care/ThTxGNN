# Azithromycin: จากยาปฏิชีวนะสู่กลุ่มอาการความหนืดเลือดสูงแบบหลายโคลน

## สรุปสั้นๆ

Azithromycin เป็นยาปฏิชีวนะกลุ่มแมโครไลด์ที่ใช้รักษาการติดเชื้อแบคทีเรียหลากหลายชนิด ทั้งทางเดินหายใจ ผิวหนัง และโรคติดต่อทางเพศสัมพันธ์ โดยมีคุณสมบัติพิเศษในการปรับสมดุลภูมิคุ้มกันนอกเหนือจากฤทธิ์ต้านเชื้อ
โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **กลุ่มอาการความหนืดเลือดสูงแบบหลายโคลน (Polyclonal Hyperviscosity Syndrome)** ด้วยคะแนน 99.81%
ปัจจุบัน **ไม่มี** การทดลองทางคลินิกหรือวรรณกรรมสนับสนุนข้อบ่งใช้นี้ และยาไม่ได้รับการจดทะเบียนในประเทศไทย

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียดจากฐานข้อมูล DrugBank ที่มีในชุดข้อมูลนี้ อย่างไรก็ตาม จากองค์ความรู้ทางเภสัชวิทยาทั่วไป Azithromycin ออกฤทธิ์ยับยั้งการสังเคราะห์โปรตีนของแบคทีเรียโดยจับกับหน่วยย่อย 50S ของไรโบโซม และนอกจากนี้ยังมีฤทธิ์ปรับสมดุลภูมิคุ้มกัน (immunomodulatory) โดยลดการผลิตไซโตไคน์ก่อการอักเสบ เช่น IL-6 และ TNF-α

กลุ่มอาการความหนืดเลือดสูงแบบหลายโคลนเกิดจากการผลิตอิมมูโนโกลบูลินมากเกินไปจากพลาสมาเซลล์หลายโคลน ในทางทฤษฎีฤทธิ์ปรับภูมิคุ้มกันของ Azithromycin อาจส่งผลต่อการหลั่งอิมมูโนโกลบูลินดังกล่าว อย่างไรก็ตาม กลไกนี้เป็นเพียงการอนุมานเชิงทฤษฎี ยังไม่มีงานวิจัยใดพิสูจน์ความเชื่อมโยงโดยตรง

**ข้อสรุปด้านความสมเหตุสมผล:** ความน่าเชื่อถือทางชีววิทยา (biological plausibility) ของการคาดการณ์นี้ต่ำมาก ไม่มีหลักฐานทางคลินิกหรือก่อนคลินิกใดสนับสนุน การคาดการณ์มาจากโมเดล TxGNN เพียงอย่างเดียว

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
การคาดการณ์นี้อยู่ในระดับ L5 ซึ่งหมายความว่ามาจากโมเดล TxGNN เพียงอย่างเดียว ไม่มีหลักฐานทางคลินิกหรือก่อนคลินิกสนับสนุน และไม่มีกลไกการออกฤทธิ์โดยตรงที่พิสูจน์แล้วระหว่าง Azithromycin กับกลุ่มอาการความหนืดเลือดสูงแบบหลายโคลน ประกอบกับยายังไม่ได้รับการจดทะเบียนในประเทศไทย

**หากต้องการดำเนินการต่อต้อง:**
- ดาวน์โหลดเอกสารกำกับยา (ใบเสร็จ/Package Insert) จาก Thai FDA เพื่อรับข้อมูลข้อบ่งใช้ที่ได้รับอนุมัติ คำเตือน และข้อห้ามใช้
- สืบค้นข้อมูล MOA โดยละเอียดจาก DrugBank API (DB00207)
- ดำเนินการศึกษาก่อนคลินิก (in vitro) เพื่อตรวจสอบผลของ Azithromycin ต่อการผลิตอิมมูโนโกลบูลินในกลุ่มเซลล์พลาสมาเซลล์
- พิจารณาเปรียบเทียบกับข้อบ่งใช้ที่มีหลักฐานสูงกว่าในชุดข้อมูลนี้ เช่น **先天性血液疾病 (Rank 10, L3)** หรือ **Septicemic Plague (Rank 9, L4)** ซึ่งมีการทดลองทางคลินิกและวรรณกรรมสนับสนุนมากกว่า

> ⚠️ **ข้อสงวนสิทธิ์**: รายงานนี้จัดทำขึ้นเพื่อการวิจัยเท่านั้น ไม่ถือเป็นคำแนะนำทางการแพทย์ ผลการทำนายจาก TxGNN ต้องผ่านการตรวจสอบทางคลินิกก่อนนำไปใช้จริง
