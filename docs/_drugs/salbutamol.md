# Salbutamol: จากโรคหืดสู่เยื่อบุตาอักเสบแบบ Papillary

## สรุปสั้นๆ

Salbutamol เป็นยาขยายหลอดลมกลุ่ม Short-Acting Beta-2 Adrenergic Agonist (SABA) ที่ใช้กันอย่างแพร่หลายในการรักษาโรคหืดและภาวะหลอดลมหดเกร็ง โดยออกฤทธิ์กระตุ้น β2-adrenergic receptor บนกล้ามเนื้อเรียบของทางเดินหายใจ
โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **เยื่อบุตาอักเสบแบบ Papillary (Papillary Conjunctivitis)** ด้วยคะแนนการทำนายสูงถึง **99.9964%**
อย่างไรก็ตาม ปัจจุบัน **ยังไม่มีการทดลองทางคลินิกหรือวรรณกรรมสนับสนุน** สำหรับข้อบ่งใช้นี้โดยตรง

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

Salbutamol ออกฤทธิ์ผ่านการกระตุ้น β2-adrenergic receptor บนกล้ามเนื้อเรียบของหลอดลม ทำให้เกิดการขยายตัวและบรรเทาอาการหดเกร็ง นอกจากนี้ β2-receptor ยังพบได้บนเซลล์มาสต์ (mast cells) เซลล์บุผิว และเซลล์ภูมิคุ้มกันต่างๆ ซึ่งเมื่อถูกกระตุ้นจะยับยั้งการปล่อยสารก่อการอักเสบ เช่น histamine และ leukotrienes ได้ในระดับหนึ่ง

เยื่อบุตาอักเสบแบบ Papillary มักมีส่วนประกอบของการอักเสบที่เกี่ยวข้องกับการกระตุ้นเซลล์มาสต์และการตอบสนองผ่าน IgE ซึ่งเป็นกลไกที่มีความคล้ายคลึงกับโรคหืดและภูมิแพ้ทางเดินหายใจ การศึกษาในสัตว์ทดลองได้แสดงให้เห็นว่า Salbutamol สามารถยับยั้งการตอบสนองแบบ allergic conjunctivitis ผ่าน β2-receptor บนเนื้อเยื่อตาได้ จึงพอมีพื้นฐานเชิงกลไกที่สนับสนุนความเป็นไปได้ของการทำนายนี้

อย่างไรก็ตาม ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียดในบริบทของ papillary conjunctivitis และความเชื่อมโยงทางคลินิกยังไม่ได้รับการพิสูจน์ การทำนายนี้อิงจากโมเดล TxGNN ซึ่งวิเคราะห์ความสัมพันธ์เชิงโครงสร้างในกราฟความรู้ทางชีวการแพทย์ขนาดใหญ่ และยังต้องการการยืนยันจากการศึกษาจริง

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
แม้คะแนน TxGNN จะสูงมาก แต่ยังไม่มีหลักฐานจากการทดลองทางคลินิกหรือวรรณกรรมสนับสนุนโดยตรง ระดับหลักฐานอยู่ที่ L5 ซึ่งหมายความว่ามีเฉพาะการทำนายจากโมเดลเท่านั้น ยังไม่เพียงพอสำหรับการดำเนินการต่อ

**หากต้องการดำเนินการต่อต้อง:**
- ทบทวนวรรณกรรมด้านการใช้ β2-agonist ในโรคตาอักเสบและการศึกษาฤทธิ์ต้านการอักเสบบนเนื้อเยื่อตา
- ออกแบบการศึกษาก่อนคลินิก (in vitro/in vivo) เพื่อทดสอบสมมติฐานกลไก β2-receptor บนเยื่อบุตา
- ตรวจสอบข้อมูลการขึ้นทะเบียนใน Thai FDA อีกครั้ง เนื่องจาก Salbutamol เป็นยาพื้นฐาน (Essential Medicine) ที่ใช้อย่างแพร่หลายทั่วโลก การที่ query ไม่พบข้อมูลอาจเป็นปัญหาเชิงเทคนิคของ pipeline
- รวบรวม MOA จาก DrugBank และแหล่งข้อมูลเภสัชวิทยาอื่นเพื่อประเมินความเชื่อมโยงกับ papillary conjunctivitis ให้ครบถ้วนก่อนพิจารณาขั้นตอนต่อไป
