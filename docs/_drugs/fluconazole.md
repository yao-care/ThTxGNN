# Fluconazole: จากการติดเชื้อรา สู่ โรคกระจกตาและเยื่อบุตาอักเสบแบบจุด

## สรุปสั้นๆ

Fluconazole เป็นยาต้านเชื้อราในกลุ่มไตรอะโซล ที่ใช้รักษาการติดเชื้อแคนดิดา (Candidiasis) และเยื่อหุ้มสมองอักเสบจากเชื้อคริปโตค็อกคัส (Cryptococcal Meningitis)
โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **โรคกระจกตาและเยื่อบุตาอักเสบแบบจุด (Punctate Epithelial Keratoconjunctivitis)**
อย่างไรก็ตาม ปัจจุบัน **ไม่มีการทดลองทางคลินิกหรือวรรณกรรมใดสนับสนุน** แนวทางนี้เลย

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียดจากแหล่งข้อมูล ตามที่ทราบ Fluconazole เป็นยาต้านเชื้อราชนิดไตรอะโซลที่ออกฤทธิ์ยับยั้งเอนไซม์ CYP51 (Lanosterol 14α-demethylase) ซึ่งเป็นขั้นตอนสำคัญในการสังเคราะห์เออร์โกสเตอรอล (Ergosterol) ผนังเซลล์เชื้อรา เมื่อเออร์โกสเตอรอลหายไป ผนังเซลล์เสียความสมบูรณ์และเชื้อราตายในที่สุด ยาได้รับการพิสูจน์ประสิทธิภาพอย่างชัดเจนในการรักษาการติดเชื้อ Candida และ Cryptococcus ทั้งในระบบและเฉพาะที่

โรคกระจกตาและเยื่อบุตาอักเสบแบบจุดมีสาเหตุหลักจากไวรัส (เช่น Adenovirus, HSV) หรือภาวะตาแห้ง (Dry Eye Syndrome) **ไม่ใช่** การติดเชื้อรา กลไกต้านเชื้อราของ Fluconazole จึงไม่มีความเชื่อมโยงโดยตรงกับพยาธิกำเนิดของโรคนี้ โดยตัวแบบเองก็ระบุว่า "ความสัมพันธ์ทางกลไกอ่อนมาก"

การทำนายนี้น่าจะเกิดจากการเชื่อมโยงทางอ้อมในกราฟความรู้ (Knowledge Graph Co-occurrence) มากกว่าจะมีกลไกทางชีววิทยาที่ชัดเจน และยังไม่มีข้อมูลทางคลินิกหรือการวิจัยก่อนคลินิกรองรับในโรคนี้เลย

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
การคาดการณ์อยู่ในระดับ L5 (มีเพียงผลจากโมเดลเท่านั้น) ไม่มีทั้งการทดลองทางคลินิกและวรรณกรรมสนับสนุน และกลไกต้านเชื้อราของ Fluconazole ไม่มีความเชื่อมโยงกับพยาธิสรีรวิทยาของโรคกระจกตาและเยื่อบุตาอักเสบแบบจุดซึ่งมีสาเหตุหลักเป็นไวรัสหรือตาแห้ง

**หากต้องการดำเนินการต่อต้อง:**
- ตรวจสอบว่ามีกรณีการติดเชื้อราในกระจกตา (Fungal Keratitis) ที่อาจเกิดภาวะแทรกซ้อนเป็น Punctate Epithelial Keratoconjunctivitis หรือไม่
- ดาวน์โหลดและวิเคราะห์เอกสารกำกับยาจาก TFDA เพื่อเติมข้อมูลคำเตือนและข้อห้ามใช้ที่ยังขาดอยู่ (DG001)
- ทบทวนกลไกการออกฤทธิ์โดยละเอียดผ่าน DrugBank API (DG002) ก่อนพิจารณาตัวบ่งชี้อันดับถัดไปที่มีหลักฐานดีกว่า เช่น Pneumocystosis (rank 7, L3) หรือ Plasmodium falciparum malaria (rank 2, L4)
