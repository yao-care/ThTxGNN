# Verapamil: จากหัวใจเต้นผิดจังหวะสู่บล็อกสาขาการนำไฟฟ้า (Obsolete Bundle Branch Block)

## สรุปสั้นๆ

Verapamil เป็น L-type calcium channel blocker ที่ใช้รักษาหัวใจเต้นผิดจังหวะ ความดันโลหิตสูง และโรคหลอดเลือดหัวใจตีบ โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **บล็อกสาขาการนำไฟฟ้า (Obsolete Bundle Branch Block)** ซึ่งเป็นคำวินิจฉัยที่ถูกยกเลิกแล้วในระบบ ontology ทางการแพทย์ และในทางคลินิก bundle branch block ถือเป็นข้อห้ามสัมพัทธ์สำหรับยานี้ ปัจจุบัน**ไม่มีการทดลองทางคลินิกหรือวรรณกรรมใดสนับสนุน**แนวทางนี้เลย

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียดจาก DrugBank อย่างไรก็ตาม จากบริบทในรายงานฉบับนี้ ทราบว่า Verapamil เป็น **L-type calcium channel blocker** ที่ออกฤทธิ์ชะลอการนำไฟฟ้าผ่านโหนด AV ลดอัตราการเต้นของหัวใจ และลดแรงหดตัวของกล้ามเนื้อหัวใจ ซึ่งเป็นกลไกที่เกี่ยวข้องโดยตรงกับการรักษาความผิดปกติของจังหวะหัวใจ

TxGNN น่าจะอนุมานความเชื่อมโยงระหว่าง Verapamil กับ bundle branch block ผ่านเครือข่ายความสัมพันธ์ระหว่างยาและโรคในกราฟชีววิทยา เนื่องจากทั้งสองสภาวะต่างเกี่ยวข้องกับระบบการนำไฟฟ้าของหัวใจห้องบน-ห้องล่าง

อย่างไรก็ตาม มีข้อพิจารณาทางคลินิกสำคัญสองประการที่ทำให้การคาดการณ์นี้มีความเสี่ยงสูง: **(1)** คำว่า "obsolete" ในชื่อโรคบ่งชี้ว่าคำวินิจฉัยนี้ถูกยกเลิกแล้วในระบบ ontology มาตรฐาน ความหมายทางคลินิกจำเป็นต้องได้รับการยืนยันใหม่ก่อนดำเนินการใดๆ **(2)** bundle branch block ที่สมบูรณ์เป็นข้อห้ามสัมพัทธ์ของ Verapamil โดยตรง เพราะยาอาจยิ่งทำให้การนำไฟฟ้าเสื่อมลงจนเกิด complete heart block ได้

## ข้อมูลการวางจำหน่ายในประเทศไทย

Verapamil **ยังไม่ได้รับการขึ้นทะเบียน** และไม่มีการวางจำหน่ายในประเทศไทย ณ วันที่ 1 พฤษภาคม 2569 ไม่มีใบอนุญาตใดๆ ที่สามารถอ้างอิงได้

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
ข้อบ่งใช้ที่คาดการณ์อันดับ 1 ใช้คำวินิจฉัยที่ถูกยกเลิกแล้วในระบบ ontology ไม่มีหลักฐานทางคลินิกหรือวรรณกรรมรองรับใดๆ และกลไกของ Verapamil มีความเสี่ยงในทางคลินิกสำหรับสภาวะนี้โดยตรง ประกอบกับยายังไม่ได้วางจำหน่ายในประเทศไทย ทำให้ยังไม่เหมาะสมที่จะดำเนินการต่อในขั้นนี้

**หากต้องการดำเนินการต่อต้อง:**
- ยืนยันนิยามทางคลินิกและ ICD code ปัจจุบันที่ตรงกับ bundle branch block ที่ต้องการศึกษา เพื่อทดแทนคำวินิจฉัยที่ถูกยกเลิก
- ดาวน์โหลดและวิเคราะห์ฉลากยาจาก TFDA หรือ EMA/FDA เพื่อรวบรวมข้อมูล MOA คำเตือน และข้อห้ามใช้ที่ครบถ้วน
- สืบค้น DrugBank API (DB00661) เพื่อข้อมูลกลไกการออกฤทธิ์อย่างเป็นทางการ
- พิจารณาข้อบ่งใช้อันดับถัดไปที่มีกลไกสอดคล้องและหลักฐานมากกว่า เช่น **malignant renovascular hypertension (rank 2, L4)** ที่มีวรรณกรรมสนับสนุน 2 ฉบับ และ Verapamil มีกลไกการลดความดันโลหิตที่เกี่ยวข้อง
