# Isotretinoin: จากสิวรุนแรงชนิดนูนสู่โรคไตจากความดันโลหิตสูงชนิดร้ายแรง

## สรุปสั้นๆ

Isotretinoin เป็นยากลุ่มเรตินอยด์สังเคราะห์ (13-cis retinoic acid) ที่ใช้รักษาสิวรุนแรงชนิดนูน (severe nodular acne) ซึ่งไม่ตอบสนองต่อการรักษาแบบอื่นมาอย่างยาวนาน
โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **โรคไตจากความดันโลหิตสูงชนิดร้ายแรง (Malignant Hypertensive Renal Disease)**
ปัจจุบัน **ไม่มีการทดลองทางคลินิก** และ **ไม่มีวรรณกรรม** สนับสนุนแนวทางนี้โดยตรง การประเมินนี้อิงเฉพาะการทำนายจากโมเดลเท่านั้น

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียด ตามข้อมูลที่ทราบ Isotretinoin เป็น 13-cis isomer ของกรดเรตินอยด์ ออกฤทธิ์ผ่านการจับกับตัวรับนิวเคลียร์ RAR (Retinoic Acid Receptor) และ RXR (Retinoid X Receptor) ซึ่งควบคุมการแสดงออกของยีนที่เกี่ยวข้องกับการแบ่งตัว การแพร่พันธุ์ และการอักเสบของเซลล์ ประสิทธิผลในการรักษาสิวรุนแรงได้รับการพิสูจน์แล้ว และในเชิงกลไกอาจใช้ได้กับโรคที่มีการอักเสบเรื้อรังและการเกิดพังผืด

กลไกที่อาจเชื่อมโยงกับโรคไตจากความดันโลหิตสูงชนิดร้ายแรง ได้แก่ การกระตุ้น RAR/RXR ซึ่งอาจมีผลยับยั้งการแพร่พันธุ์ของเซลล์เมแซงเจียมในไต (renal mesangial cells) และปรับเส้นทางการเกิดพังผืดผ่าน TGF-β ซึ่งเป็นกลไกสำคัญในการเสื่อมของไตจากความดันโลหิตสูง อย่างไรก็ตาม ความเชื่อมโยงนี้ยังเป็นเพียงทฤษฎีที่ยังไม่ผ่านการทดสอบในแบบจำลองโรค

การคาดการณ์นี้มาจากโมเดล TxGNN ซึ่งใช้เครือข่ายความรู้ทางชีวการแพทย์ขนาดใหญ่ในการหาความสัมพันธ์ระหว่างยาและโรค ยังไม่มีการศึกษาก่อนคลินิกหรือคลินิกใดๆ ที่ทดสอบ isotretinoin ในข้อบ่งใช้นี้โดยตรง จึงต้องใช้ความระมัดระวังในการตีความ

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
หลักฐานอยู่ในระดับ L5 — มีเฉพาะการทำนายจากโมเดล TxGNN โดยไม่มีการทดลองทางคลินิก วรรณกรรม หรือการศึกษาก่อนคลินิกใดๆ สนับสนุนการใช้ isotretinoin สำหรับโรคไตจากความดันโลหิตสูงชนิดร้ายแรง ประกอบกับยายังไม่ได้ขึ้นทะเบียนในประเทศไทย และข้อมูลความปลอดภัยในบริบทนี้ยังขาดอยู่มาก

**หากต้องการดำเนินการต่อต้อง:**
- รวบรวมข้อมูลกลไกการออกฤทธิ์ (MOA) ที่ครบถ้วนจาก DrugBank API
- ดาวน์โหลดและวิเคราะห์เอกสารกำกับยา (เพื่อให้ได้คำเตือนหลัก ข้อห้ามใช้ และปฏิกิริยาระหว่างยา)
- ค้นหาการศึกษาก่อนคลินิก (in vitro / in vivo) ในแบบจำลองโรคไตจากความดันโลหิตสูงที่เกี่ยวข้องกับ retinoid pathway
- ประเมินความเป็นไปได้ในการนำยาเข้ามาจดทะเบียนในประเทศไทย หรือใช้กรอบ Named Patient / Compassionate Use ก่อนหากจำเป็น
