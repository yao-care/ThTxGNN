# Timolol: จากต้อหินชนิดมุมเปิดสู่ต้อหินทางพันธุกรรมปฐมภูมิ

## สรุปสั้นๆ

Timolol เป็นยาบล็อกเบต้า (β-blocker) ชนิดไม่เลือก ที่ใช้กันอย่างแพร่หลายมาหลายทศวรรษในการรักษาต้อหินชนิดมุมเปิดและลดความดันในลูกตา โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **ต้อหินทางพันธุกรรมปฐมภูมิ (Primary Hereditary Glaucoma)** ผ่านกลไกการยับยั้งการผลิตน้ำในลูกตาผ่านตัวรับ β2 ที่ ciliary body อย่างไรก็ตาม ปัจจุบันหลักฐานสนับสนุนยังอ่อนมาก มีเพียง **การทดลองทางคลินิก 1 รายการ** ที่มีความเกี่ยวข้องทางอ้อม และ **ยังไม่มีวรรณกรรมที่เฉพาะเจาะจง** กับ hereditary glaucoma รองรับแนวทางนี้โดยตรง

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

Timolol ออกฤทธิ์โดยจับกับตัวรับ β2-adrenergic ที่เซลล์ epithelium ไม่มีเม็ดสี (non-pigmented epithelium) ของ ciliary body ในลูกตา ส่งผลให้การผลิตน้ำในลูกตา (aqueous humor) ลดลงประมาณ 20–35% และความดันในลูกตาลดลงตามมา กลไกนี้เป็นรากฐานสำคัญของการรักษาต้อหินทุกชนิดที่มีความดันในลูกตาสูงเป็นปัจจัยเสี่ยงหลัก

ต้อหินทางพันธุกรรมปฐมภูมิ เช่น กรณีที่มีการกลายพันธุ์ของยีน MYOC หรือ OPTN มีพยาธิสภาพหลักอยู่ที่การระบายน้ำในลูกตาบกพร่อง ซึ่งนำไปสู่ความดันในลูกตาสูงเรื้อรังและความเสียหายของขั้วประสาทตาในที่สุด ในเชิงกลไก การลดการผลิตน้ำในลูกตาด้วย Timolol จึงอาจช่วยชดเชยการระบายที่บกพร่องได้ — เป็นแนวทางที่สมเหตุสมผลในเชิงทฤษฎี

อย่างไรก็ตาม ยังขาดการศึกษาทางคลินิกที่เน้นเฉพาะกลุ่มผู้ป่วยที่ยืนยันการกลายพันธุ์ทางพันธุกรรมโดยตรง คะแนน TxGNN สูงถึง 98.64% น่าจะสะท้อนความคล้ายคลึงกันเชิงโครงสร้างในฐานข้อมูล Knowledge Graph ระหว่างต้อหินทั่วไปกับ hereditary glaucoma มากกว่าการมีหลักฐานทางคลินิกสำหรับ subtype นี้โดยเฉพาะ

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
หลักฐานสนับสนุนการใช้ Timolol เฉพาะเจาะจงในต้อหินทางพันธุกรรมปฐมภูมิยังอยู่ในระดับ L4 โดยมีเพียงหนึ่งการทดลองที่มีความเกี่ยวข้องทางอ้อมและไม่มีวรรณกรรมโดยตรง แม้กลไกทางทฤษฎีจะสมเหตุสมผล แต่ยังขาดหลักฐานทางคลินิกที่ยืนยันในกลุ่มผู้ป่วยพันธุกรรม

**หากต้องการดำเนินการต่อต้อง:**
- ค้นหาการศึกษาในผู้ป่วยต้อหินที่มีการยืนยันการกลายพันธุ์ทางพันธุกรรม (MYOC, OPTN, TBK1, LTBP2, CYP1B1)
- ดึงข้อมูลกลไกการออกฤทธิ์ (MOA) โดยละเอียดจาก DrugBank API (แก้ไข DG002)
- ดาวน์โหลดและวิเคราะห์เอกสารกำกับยาจาก Thai FDA เพื่อประเมินคำเตือนและข้อห้ามใช้ (แก้ไข DG001)
- พิจารณาออกแบบการศึกษานำร่องในกลุ่มผู้ป่วยต้อหินทางพันธุกรรมที่ได้รับการยืนยันโดยเฉพาะ
