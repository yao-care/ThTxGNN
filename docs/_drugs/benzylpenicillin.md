<h3>為什麼這個預測合理？</h3>

<p>牙冠周炎是智齒萌出不全時周圍軟組織的感染，常見致病菌包括：</p>
<ul>
<li>厭氧菌：Peptococcus, Peptostreptococcus, Bacteroides, Fusobacterium</li>
<li>需氧菌：Streptococcus 等</li>

</ul>
<p>Penicillins (包括 benzylpenicillin 及其口服衍生物) 對這些病原菌有效，是口腔感染的一線抗生素選擇。</p>

## 安全性考量
- **過敏反應**：青黴素類過敏史為絕對禁忌
- **交叉過敏**：與其他 beta-lactam 類藥物可能有交叉反應
- **抗藥性考量**：部分口腔菌株產生 beta-lactamase，可能需要合併使用 beta-lactamase 抑制劑或改用 amoxicillin-clavulanate
- **腎功能不全**：需調整劑量

## 結論與下一步
**決策：Go**
**理由：** TxGNN 的預測獲得充分的文獻和臨床實務支持。Penicillins 類藥物已是口腔感染（包括牙冠周炎）的一線抗生素選擇。泰國臨床實務中，amoxicillin 是更常用的口服選擇，benzylpenicillin 則用於需要注射給藥的嚴重感染。
**若要推進需要：**
1. 臨床上優先考慮口服 amoxicillin 或 penicillin V，而非注射型 benzylpenicillin
2. 對於嚴重牙冠周炎合併蜂窩性組織炎，可考慮注射型 benzylpenicillin
3. 考慮合併 metronidazole 以涵蓋厭氧菌
4. 注意青黴素過敏病史，必要時改用 clindamycin 或 azithromycin
5. 此預測更多確認現有實務，而非提出新的治療方向
