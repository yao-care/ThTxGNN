# Indomethacin: จากยาต้านการอักเสบ (NSAID) สู่กลุ่มอาการ Brachydactyly-Syndactyly

## สรุปสั้นๆ

Indomethacin เป็นยาต้านการอักเสบชนิดไม่ใช่สเตียรอยด์ (NSAID) ที่ออกฤทธิ์ผ่านการยับยั้งเอนไซม์ COX-1 และ COX-2 ซึ่งใช้รักษาโรคข้ออักเสบและอาการปวดอักเสบมายาวนาน แต่ปัจจุบันยังไม่ได้รับการขึ้นทะเบียนในประเทศไทย
โมเดล TxGNN ให้คะแนนสูงสุดแก่ **กลุ่มอาการ Brachydactyly-Syndactyly** ซึ่งเป็นความผิดปกติทางพันธุกรรมด้านการพัฒนาของนิ้วมือและนิ้วเท้า โดยอาศัยการเชื่อมโยงผ่านวิถีสัญญาณ PGE2 ในกระบวนการสร้างแขนขา
ปัจจุบัน **ยังไม่มีการทดลองทางคลินิกหรือวรรณกรรมใดสนับสนุน** แนวทางนี้ หลักฐานอยู่ในระดับ L5 (การทำนายจากโมเดลเท่านั้น)

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียดจากชุดข้อมูลนี้ ตามที่ทราบกันโดยทั่วไป Indomethacin เป็น NSAID ที่ยับยั้งเอนไซม์ COX-1 และ COX-2 ส่งผลให้ระดับ PGE2 (Prostaglandin E2) และ TXA2 (Thromboxane A2) ในเนื้อเยื่อลดลง กลไกนี้เป็นพื้นฐานของฤทธิ์ต้านการอักเสบ บรรเทาปวด และลดไข้

กลุ่มอาการ Brachydactyly-Syndactyly เกี่ยวข้องกับความผิดปกติของการพัฒนาแขนขาในระยะตัวอ่อน โดย PGE2 มีบทบาทในวิถีสัญญาณที่ควบคุมการสร้างรูปแบบแขนขา (limb patterning) ทำให้โมเดล TxGNN สร้างการเชื่อมโยงระหว่าง Indomethacin กับกลุ่มอาการนี้ในเชิงทฤษฎี

อย่างไรก็ตาม การเชื่อมโยงนี้เป็นการอนุมานเชิงโมเดลที่ยังขาดหลักฐานสนับสนุน กลุ่มอาการนี้มีสาเหตุหลักจากการกลายพันธุ์ทางพันธุกรรม ซึ่งการลด PGE2 ด้วย Indomethacin ไม่น่าจะแก้ไขพยาธิกำเนิดหลักได้ การเชื่อมโยงจึงอยู่ในระดับสูงมากและต้องการหลักฐานพรีคลินิกยืนยันก่อนที่จะพิจารณาขั้นตอนถัดไป

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
การทำนายอยู่ที่ระดับ L5 มีเพียงคะแนนจากโมเดล TxGNN โดยไม่มีหลักฐานทางคลินิกหรือพรีคลินิกรองรับ ประกอบกับ Indomethacin ยังไม่ได้รับการขึ้นทะเบียนในประเทศไทย และข้อมูลความปลอดภัย (MOA, คำเตือน, ข้อห้ามใช้) ยังไม่ครบถ้วน

**หากต้องการดำเนินการต่อต้อง:**
- ขอข้อมูลกลไกการออกฤทธิ์ (MOA) โดยละเอียดจาก DrugBank API
- ดาวน์โหลดและวิเคราะห์เอกสารกำกับยาจาก Thai FDA เพื่อประเมินคำเตือนและข้อห้ามใช้ (Blocking gap: DG001)
- ทบทวนหลักฐานพรีคลินิกที่แสดงผลของ Indomethacin ต่อวิถีสัญญาณ PGE2 ในกระบวนการสร้างแขนขา (limb morphogenesis)
- พิจารณาข้อบ่งใช้อันดับที่ 8 (Juvenile Idiopathic Arthritis) ซึ่งมีหลักฐานระดับ L2 พร้อม RCT (PMID [362571](https://pubmed.ncbi.nlm.nih.gov/362571/)) และคำแนะนำ "Proceed with Guardrails" ซึ่งมีความพร้อมในการพัฒนาต่อมากกว่าข้อบ่งใช้อันดับที่ 1 อย่างมีนัยสำคัญ
