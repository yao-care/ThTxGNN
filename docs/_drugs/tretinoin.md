# Tretinoin: จากสิวและมะเร็งเม็ดเลือดขาว APL สู่ Rheumatoid Nodulosis

## สรุปสั้นๆ

Tretinoin (กรดวิตามิน A ชนิด all-trans retinoic acid, ATRA) เป็นยาในกลุ่ม Retinoid ที่รู้จักกันดีสำหรับการรักษาสิว (Acne Vulgaris) และมะเร็งเม็ดเลือดขาวชนิด Acute Promyelocytic Leukemia (APL)
โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **Rheumatoid Nodulosis** (ภาวะปมรูมาตอยด์ที่เกิดในโรคข้ออักเสบรูมาตอยด์)
ปัจจุบัน **ไม่พบการทดลองทางคลินิกหรือวรรณกรรม** ที่สนับสนุนแนวทางนี้โดยตรง ข้อมูลทั้งหมดมาจากการทำนายของโมเดลเพียงแหล่งเดียว

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

Tretinoin ออกฤทธิ์ผ่านการกระตุ้น Retinoic Acid Receptor (RAR) ซึ่งเป็น nuclear receptor ที่ควบคุมการแสดงออกของยีนสำคัญที่เกี่ยวข้องกับการแบ่งตัวและการตอบสนองของเซลล์ภูมิคุ้มกัน ผลที่ทราบคือ Tretinoin สามารถยับยั้งเส้นทาง NF-κB ซึ่งเป็นตัวกลางหลักของการอักเสบ จึงเกิดแนวคิดว่าอาจนำมาใช้รักษาโรคที่มีพื้นฐานจากการอักเสบผ่านภูมิคุ้มกัน

Rheumatoid Nodulosis เป็นรูปแบบหนึ่งของโรคข้ออักเสบรูมาตอยด์ที่มีก้อนรูมาตอยด์ขนาดใหญ่จำนวนมาก มักกลับเป็นซ้ำ และพยาธิสภาพเกิดจากการตอบสนองของ T cell ร่วมกับ cytokines เช่น TNF-α และ IL-6 ซึ่งทั้งหมดนี้อาจถูกปรับเปลี่ยนได้ผ่านเส้นทาง RAR/NF-κB ตามที่โมเดล TxGNN อนุมาน

อย่างไรก็ตาม ต้องย้ำว่าข้อสันนิษฐานนี้เป็นเพียงการอนุมานเชิงกลไกล้วนๆ และ **ไม่มีหลักฐานทางคลินิกหรือก่อนคลินิกใดๆ** สำหรับ Tretinoin ใน Rheumatoid Nodulosis ทั้งสิ้น ข้อมูลกลไกการออกฤทธิ์โดยละเอียดจาก DrugBank ยังคงรอการเติมเต็ม และจำเป็นต้องศึกษาเพิ่มก่อนประเมินต่อ

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
ระดับหลักฐานอยู่ที่ L5 ซึ่งหมายความว่าการสนับสนุนมาจากการทำนายของโมเดล TxGNN เพียงอย่างเดียว ไม่มีการศึกษาทางคลินิกหรือก่อนคลินิกใดๆ รองรับข้อบ่งใช้นี้ ประกอบกับยายังไม่ได้รับการขึ้นทะเบียนในประเทศไทย และยังขาดข้อมูล MOA และความปลอดภัยที่จำเป็นสำหรับการประเมินขั้นต่อไป

**หากต้องการดำเนินการต่อต้อง:**
- ดึงข้อมูลกลไกการออกฤทธิ์ (MOA) โดยละเอียดจาก DrugBank API (แก้ไข DG002)
- ดาวน์โหลดและวิเคราะห์ฉลากยา/คำเตือนจาก TFDA เพื่อประเมินความปลอดภัยเบื้องต้น (แก้ไข DG001)
- ทบทวนวรรณกรรมเพิ่มเติมเกี่ยวกับ Retinoid ในโรคข้ออักเสบรูมาตอยด์และ nodulosis
- ออกแบบการศึกษาก่อนคลินิก (in vitro / in vivo) เพื่อยืนยันผลของ Tretinoin ต่อ rheumatoid nodule model ก่อนพิจารณาเข้าสู่ขั้นตอนทางคลินิก
