# Metoprolol: จากโรคหัวใจและความดันโลหิตสูงสู่โรคไตจากความดันโลหิตสูงชนิดร้ายแรง

## สรุปสั้นๆ

Metoprolol เป็นยากลุ่ม beta-1 selective blocker ที่ใช้รักษาความดันโลหิตสูง โรคหัวใจขาดเลือด และภาวะหัวใจล้มเหลวมาหลายทศวรรษทั่วโลก แม้ปัจจุบันยังไม่มีการจดทะเบียนในประเทศไทย
โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **โรคไตจากความดันโลหิตสูงชนิดร้ายแรง (Malignant Hypertensive Renal Disease)**
อย่างไรก็ตาม ปัจจุบันยังไม่มีการทดลองทางคลินิกหรือวรรณกรรมที่สนับสนุนแนวทางนี้โดยตรง หลักฐานอยู่ในระดับ **L5** ซึ่งเป็นการทำนายจากโมเดลเพียงอย่างเดียว

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันฐานข้อมูล DrugBank ยังไม่ส่งคืนข้อมูลกลไกการออกฤทธิ์ (MOA) ผ่าน API สำหรับการวิเคราะห์ครั้งนี้ อย่างไรก็ตาม จากวรรณกรรมทางเภสัชวิทยาที่เป็นที่ยอมรับอย่างกว้างขวาง Metoprolol ออกฤทธิ์โดยการแข่งขันยับยั้งตัวรับ β1-adrenergic ที่หัวใจและไตอย่างจำเพาะ ส่งผลให้อัตราการเต้นหัวใจลดลง แรงบีบตัวหัวใจลดลง และยับยั้งการหลั่ง renin จากไต ซึ่งรวมกันทำให้ความดันโลหิตลดลงผ่านการกดระบบ RAAS

โรคไตจากความดันโลหิตสูงชนิดร้ายแรงเป็นภาวะฉุกเฉินที่ความดันโลหิตสูงรุนแรงมากจนทำให้หลอดเลือดฝอยในไตเสียหายอย่างเฉียบพลัน ในเชิงทฤษฎี กลไกการลด renin และลดความดันโลหิตของ Metoprolol อาจช่วยชะลอความเสียหายที่ไตได้ มีความเชื่อมโยงเชิงกลไกในระดับหนึ่งระหว่างข้อบ่งใช้เดิม (ควบคุมความดันโลหิต) กับข้อบ่งใช้ที่ทำนาย (ปกป้องไตจากความดันโลหิตสูง)

อย่างไรก็ตาม แนวทางการรักษาภาวะนี้ในปัจจุบันมุ่งเน้นการลดความดันโลหิตอย่างรวดเร็วและแม่นยำด้วยยาฉีดเข้าหลอดเลือดดำ เช่น labetalol, nicardipine หรือ nitroprusside ยา beta-blocker ชนิดรับประทานอย่าง Metoprolol ไม่ใช่ตัวเลือกมาตรฐานในสถานการณ์ฉุกเฉินนี้ การคาดการณ์จาก TxGNN จึงน่าจะสะท้อนความใกล้ชิดทางโครงข่ายในกราฟความรู้มากกว่าการสนับสนุนทางคลินิกที่ชัดเจน

## ข้อพิจารณาด้านความปลอดภัย

กรุณาดูข้อมูลความปลอดภัยในเอกสารกำกับยา

> **หมายเหตุ:** ข้อมูลคำเตือน ข้อห้ามใช้ และปฏิกิริยาระหว่างยาจากฐาน TFDA ยังไม่สามารถดึงมาได้ในรอบนี้ ควรดาวน์โหลดเอกสารกำกับยา PDF จากเว็บไซต์ TFDA และตรวจสอบก่อนนำไปใช้ทางคลินิก

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
ไม่มีหลักฐานทางคลินิกหรือวรรณกรรมใดที่สนับสนุนการใช้ Metoprolol เฉพาะเจาะจงสำหรับโรคไตจากความดันโลหิตสูงชนิดร้ายแรง และมาตรฐานการรักษาภาวะฉุกเฉินนี้ต้องการยาฉีดที่ออกฤทธิ์รวดเร็ว ซึ่งไม่ใช่จุดแข็งของ Metoprolol ชนิดรับประทาน

**หากต้องการดำเนินการต่อต้อง:**
- ดึงข้อมูล MOA โดยละเอียดจาก DrugBank API (DG002)
- ดาวน์โหลดเอกสารกำกับยาจาก TFDA เพื่อประเมินคำเตือนและข้อห้ามใช้ (DG001)
- ทบทวนวรรณกรรมเชิงกลไกว่า beta-blocker อาจมีบทบาทในการป้องกันไตระยะยาวในผู้ป่วยที่ผ่านวิกฤต malignant hypertension หรือไม่
- พิจารณาออกแบบการศึกษาเชิงสังเกตในผู้ป่วยที่ได้รับ Metoprolol ภายหลังเหตุการณ์ malignant hypertension เพื่อประเมิน renal outcome ก่อนขยับไปสู่การทดลองทางคลินิก
