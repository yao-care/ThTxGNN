# Metronidazole: จากยาต้านโปรโตซัวสู่ Pneumocystosis

## สรุปสั้นๆ

Metronidazole เป็นยาปฏิชีวนะกลุ่ม nitroimidazole ที่ใช้รักษาการติดเชื้อโปรโตซัว (อะมีบา ไตรโคโมนัส ไจอาเดีย) และแบคทีเรียไร้อากาศ (anaerobic bacteria) มาอย่างยาวนาน โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **Pneumocystosis** (การติดเชื้อ Pneumocystis jirovecii) ด้วยคะแนน 99.99% แต่หลักฐานสนับสนุนโดยตรงยังอ่อนแอ — ไม่พบการทดลองทางคลินิกที่เกี่ยวข้อง และวรรณกรรมที่มีอยู่เชื่อมโยง Metronidazole กับ pneumocystosis เฉพาะในบริบทผู้ป่วย AIDS ที่มีการติดเชื้อซ้อน ไม่ใช่การรักษา PCP โดยตรง

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียดจาก DrugBank จากความรู้ที่มีอยู่ Metronidazole เป็นยา nitroimidazole ที่ออกฤทธิ์โดยรับอิเล็กตรอนจาก ferredoxin ในสภาวะไร้ออกซิเจน เกิดเป็นอนุมูลอิสระที่ทำลาย DNA ของจุลชีพเป้าหมาย กลไกนี้ต้องการสภาวะ anaerobic/microaerophilic จึงจะทำงานได้ ประสิทธิผลในการรักษาการติดเชื้อโปรโตซัวและแบคทีเรียไร้อากาศได้รับการพิสูจน์แล้ว

อย่างไรก็ตาม ความเชื่อมโยงกับ pneumocystosis มีข้อจำกัดที่สำคัญ Pneumocystis jirovecii เป็นเชื้อราที่ต้องการออกซิเจน Metronidazole ไม่มีฤทธิ์ฆ่าเชื้อราโดยตรง และกลไกการออกฤทธิ์ไม่รองรับการนำมาต่อต้าน P. jirovecii มาตรฐานการรักษา PCP คือ TMP-SMX (trimethoprim-sulfamethoxazole) หรือ pentamidine ไม่ใช่ nitroimidazole

สัญญาณที่ TxGNN ตรวจจับได้อาจเกิดจาก confounding: Metronidazole ปรากฏในเวชระเบียนผู้ป่วย AIDS ที่มีการติดเชื้อซ้อนหลายชนิด ผู้ป่วยกลุ่มนี้ได้รับ Metronidazole รักษา amebiasis หรือ anaerobic infection ในเวลาเดียวกับที่พัฒนา pneumocystosis จาก HIV-related immunosuppression ซึ่งเป็นเหตุการณ์ที่เกิดพร้อมกัน ไม่ใช่ความสัมพันธ์เชิงสาเหตุ-ผล

## หลักฐานจากการทดลองทางคลินิก

ปัจจุบันยังไม่มีการลงทะเบียนการทดลองทางคลินิกที่เกี่ยวข้องกับการใช้ Metronidazole ในการรักษา Pneumocystosis

> การสืบค้นพบ 23 รายการ แต่ทั้งหมดเป็นการศึกษาด้านการจัดการโรคเรื้อรัง (เบาหวาน ความเจ็บปวด สมองเสื่อม) การดูแลสุขภาพเบื้องต้น และการปลูกถ่ายอวัยวะ ไม่มีรายการใดเกี่ยวข้องกับ Metronidazole หรือ Pneumocystosis

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
กลไกการออกฤทธิ์ของ Metronidazole ซึ่งต้องอาศัยสภาวะไร้ออกซิเจนในการถูกกระตุ้น ไม่รองรับการนำมาใช้ต่อต้าน Pneumocystis jirovecii ที่เป็นเชื้อราต้องการออกซิเจน อีกทั้งไม่มีการทดลองทางคลินิกหรือหลักฐานเชิงกลไกที่สนับสนุนโดยตรง สัญญาณที่ TxGNN ตรวจจับน่าจะเกิดจาก confounding ของ co-infection pattern ในผู้ป่วย AIDS

**หากต้องการดำเนินการต่อต้อง:**
- รับข้อมูล MOA โดยละเอียดจาก DrugBank API (DG002) เพื่อยืนยันว่าไม่มีกลไกรอง (secondary mechanism) ที่อาจเกี่ยวข้องกับ P. jirovecii
- ดาวน์โหลดเอกสารกำกับยาจาก TFDA เพื่อตรวจสอบคำเตือนและข้อห้ามใช้ (DG001)
- ทบทวนว่า TxGNN prediction signal เกิดจาก network proximity หรือ co-occurrence artifact ในกราฟ
- พิจารณา indication อื่นที่มีหลักฐานแข็งแกร่งกว่าและมีกลไกชัดเจน เช่น **cap polyposis** (Rank 9, L3, Proceed with Guardrails) ซึ่ง Metronidazole มีกลไกทั้งการกำจัด H. pylori และฤทธิ์ต้านการอักเสบที่ได้รับการบันทึกในวรรณกรรม
