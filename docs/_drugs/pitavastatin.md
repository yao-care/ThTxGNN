# Pitavastatin 藥師筆記

## 一句話總結

<p class="key-answer" data-question="Pitavastatin 可以用於治療什麼新適應症？">
Pitavastatin 為 HMG-CoA 還原酶抑制劑，TxGNN 預測對家族性高膽固醇血症等脂質代謝疾病有療效，已有豐富臨床試驗及文獻支持，部分預測與現有適應症重疊。
</p>

<h3>為什麼這個預測合理？</h3>

<p>Pitavastatin 作為 statin 類藥物，其預測的新適應症在機轉上完全合理：</p>

<ol>
<li><strong>同型合子家族性高膽固醇血症 (Homozygous FH)</strong>：Pitavastatin 抑制 HMG-CoA 還原酶，減少肝臟膽固醇合成，是 FH 的一線治療選擇</li>
<li><strong>高脂蛋白血症 (Hyperlipoproteinemia)</strong>：Statin 類藥物可有效降低 LDL-C，對各類高脂蛋白血症均有療效</li>
<li><strong>家族性高膽固醇血症</strong>：已是泰國核准適應症的一部分，預測與現有適應症重疊</li>
</ol>
