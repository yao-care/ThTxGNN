# Albendazole: จากยาถ่ายพยาธิลำไส้สู่โรคพยาธิถุงลม (Alveolar Echinococcosis)

---

## สรุปสั้นๆ

Albendazole เป็นยาถ่ายพยาธิกลุ่ม benzimidazole ที่ใช้รักษาโรคพยาธิตัวกลมในลำไส้ในระดับสากล แต่ปัจจุบันยังไม่มีทะเบียนยาในประเทศไทย
โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **โรคพยาธิถุงลม (Alveolar Echinococcosis)** ซึ่งเป็นโรคปรสิตร้ายแรงที่เกิดจากพยาธิตัวตืด *Echinococcus multilocularis*
ปัจจุบันมี **การทดลองทางคลินิก 5 รายการ** และ **วรรณกรรม 20 ฉบับ** สนับสนุนแนวทางนี้

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

Albendazole ออกฤทธิ์โดยยับยั้งการรวมตัวของ β-tubulin ในเซลล์ปรสิต ซึ่งเป็นโปรตีนโครงสร้างที่จำเป็นต่อการสร้างไมโครทิวบูล ส่งผลให้ปรสิตไม่สามารถดูดซึมกลูโคสและเกิดการหยุดชะงักของกระบวนการเมตาบอลิซึมพลังงาน กลไกนี้มีฤทธิ์ครอบคลุมพยาธิตัวตืดและพยาธิตัวกลมหลายชนิด

โรคพยาธิถุงลม (Alveolar Echinococcosis) เกิดจากระยะตัวอ่อน (metacestode) ของพยาธิตัวตืด *Echinococcus multilocularis* ซึ่งมักเกิดรอยโรคในตับลักษณะคล้ายเนื้อร้าย กลไกการยับยั้ง β-tubulin ของ albendazole สามารถรบกวนการเจริญเติบโตของระยะตัวอ่อนนี้ได้โดยตรง ทำให้ยานี้เป็น **ยามาตรฐานที่แนะนำในระดับสากล** โดย WHO-IWGE สำหรับการรักษาโรคพยาธิถุงลม

อย่างไรก็ตาม albendazole มีฤทธิ์เพียง **parasitostatic** (ยับยั้งการเจริญเติบโตแต่ไม่ฆ่าปรสิต) ดังนั้นผู้ป่วยมักต้องได้รับยาเป็นระยะเวลานานหรือตลอดชีวิต การวิจัยเพื่อพัฒนาสูตรยาที่มีประสิทธิภาพมากขึ้น เช่น albendazole nanocrystals หรือการใช้ร่วมกับยาอื่น กำลังดำเนินอยู่

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Proceed with Guardrails**

**เหตุผล:**
Albendazole เป็นยามาตรฐานที่ WHO แนะนำสำหรับรักษาโรคพยาธิถุงลม (AE) อยู่แล้วในระดับสากล มีการทดลองทางคลินิก Phase 2 ขนาด n=194 ที่เสร็จสมบูรณ์ และวรรณกรรมจำนวนมากสนับสนุนประสิทธิภาพ อย่างไรก็ตาม ยานี้ยังไม่มีทะเบียนในประเทศไทย ซึ่งเป็นอุปสรรคสำคัญต่อการเข้าถึง

**หากต้องการดำเนินการต่อต้อง:**
- ดำเนินการขึ้นทะเบียนยา Albendazole กับ อย. หรือสำรวจช่องทางการนำเข้ายาพิเศษ
- รวบรวมข้อมูลคำเตือน ข้อห้ามใช้ และข้อมูลความปลอดภัยจากเอกสารกำกับยา (ปัจจุบันเป็น Data Gap ระดับ Blocking)
- ยืนยันกลไกการออกฤทธิ์โดยละเอียดจาก DrugBank API (Data Gap ระดับ High)
- จัดทำแผนการติดตามความปลอดภัยระยะยาว เนื่องจากผู้ป่วย AE มักต้องได้รับยาเป็นเวลานานหรือตลอดชีวิต
- พิจารณาความเป็นไปได้ในการศึกษาสูตรยาใหม่ที่เพิ่มการดูดซึม (เช่น nanocrystal formulation)
