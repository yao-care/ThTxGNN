# Estradiol: จากฮอร์โมนทดแทนสตรีสู่อาการ Fragile X Syndrome ในหญิงพาหะ

## สรุปสั้นๆ

Estradiol เป็นฮอร์โมนเอสโตรเจนธรรมชาติที่ใช้รักษาภาวะขาดฮอร์โมนเพศหญิง เช่น อาการหมดประจำเดือน (menopause) และภาวะต่อมเพศทำงานบกพร่อง (hypogonadism)
โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **อาการ Fragile X Syndrome ในหญิงพาหะ (Symptomatic Form of Fragile X Syndrome in Female Carrier)**
ปัจจุบัน**ไม่มีการทดลองทางคลินิกหรือวรรณกรรมใดสนับสนุนแนวทางนี้เลย** — การทำนายอยู่ในระดับ L5 ซึ่งเป็นระดับหลักฐานต่ำสุด

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์ (MOA) โดยละเอียดจาก DrugBank ตามที่ทราบ Estradiol เป็นฮอร์โมนเอสโตรเจนหลักในสตรีวัยเจริญพันธุ์ ออกฤทธิ์ผ่านตัวรับเอสโตรเจน ERα และ ERβ ซึ่งพบได้ทั้งในระบบสืบพันธุ์และระบบประสาทส่วนกลาง โดยมีคุณสมบัติในการปกป้องระบบประสาท (neuroprotection) ที่ได้รับการศึกษาในหลายบริบท

Fragile X Syndrome เกิดจากการปิดเงียบยีน FMR1 บน X chromosome ทำให้ขาดโปรตีน FMRP ซึ่งมีบทบาทสำคัญต่อ synaptic plasticity ในหญิงพาหะแบบ heterozygous จะมีการผลิต FMRP เพียงบางส่วน ส่งผลให้เกิดอาการทางระบบประสาทในระดับที่หลากหลาย Estradiol อาจมีศักยภาพในการปรับสมดุล glutamate/GABA signaling ที่ผิดปกติผ่านตัวรับในสมอง อย่างไรก็ตาม สมมติฐานนี้ยังไม่เคยได้รับการทดสอบในสัตว์ทดลองหรือมนุษย์

การทำนายของ TxGNN น่าจะสะท้อนความใกล้ชิดเชิงกราฟระหว่างเครือข่ายฮอร์โมนเอสโตรเจนกับเส้นทาง FMR1 ในฐานข้อมูลความรู้ มากกว่าจะชี้ถึงความสัมพันธ์ทางการรักษาที่พิสูจน์ได้จริง จำเป็นต้องมีการพิสูจน์เชิงกลไกก่อนดำเนินการขั้นต่อไป

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
การคาดการณ์อยู่ที่ระดับหลักฐาน L5 คือมีเพียงโมเดล TxGNN เป็นแหล่งอ้างอิงเท่านั้น ไม่มีการทดลองทางคลินิกหรือวรรณกรรมใดสนับสนุนการใช้ Estradiol สำหรับ Fragile X Syndrome ในหญิงพาหะ อีกทั้งยังขาดข้อมูลกลไกการออกฤทธิ์และข้อมูลความปลอดภัยที่จำเป็นต่อการประเมินขั้น S1

**หากต้องการดำเนินการต่อต้อง:**
- ศึกษากลไกการออกฤทธิ์ของ Estradiol ต่อ FMR1/FMRP pathway ในระดับ in vitro ก่อนเป็นลำดับแรก
- ดำเนินการศึกษาก่อนคลินิกในแบบจำลองสัตว์ทดลอง Fragile X (เช่น Fmr1 knockout mouse model)
- ดาวน์โหลดและวิเคราะห์คำเตือนและข้อห้ามใช้จากเอกสารกำกับยา Thai FDA (TFDA)
- ขอข้อมูล MOA และ DrugBank categories ที่อัปเดตล่าสุดผ่าน DrugBank API เพื่อประเมิน cytotoxicity profile และปฏิกิริยาระหว่างยา

> ⚠️ **หมายเหตุ**: รายงานนี้จัดทำขึ้นเพื่อการวิจัยเท่านั้น ไม่ถือเป็นคำแนะนำทางการแพทย์ การนำยาเก่ามาใช้ใหม่จำเป็นต้องผ่านกระบวนการตรวจสอบทางคลินิกก่อนนำไปประยุกต์ใช้จริง
