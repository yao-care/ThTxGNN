# Pyrazinamide: จากวัณโรคสู่การติดเชื้อหูชั้นกลาง

## สรุปสั้นๆ

Pyrazinamide เป็นยาต้านเชื้อวัณโรคที่ใช้มาอย่างยาวนานในฐานะส่วนหนึ่งของสูตรการรักษาวัณโรคมาตรฐาน (HRZE) โดยออกฤทธิ์เฉพาะเจาะจงต่อ *Mycobacterium tuberculosis* โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **การติดเชื้อหูชั้นกลาง (Infectious Otitis Media)** อย่างไรก็ตาม ปัจจุบัน**ยังไม่มีการทดลองทางคลินิกหรือวรรณกรรมสนับสนุน**แนวทางนี้โดยตรง หลักฐานจึงอยู่ในระดับ L5 ซึ่งต่ำที่สุด

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียด ตามที่ทราบ Pyrazinamide เป็นยาต้านวัณโรคที่มีกลไกเฉพาะเจาะจงต่อ *Mycobacterium tuberculosis* โดยสมมติฐานหลักชี้ว่ายาถูกเปลี่ยนเป็น Pyrazinoic acid ในสภาวะกรดภายในแมคโครฟาจ จากนั้นยับยั้งการสังเคราะห์กรดไขมัน (FAS I) ของเชื้อ ส่งผลให้กลไกนี้มีความเฉพาะเจาะจงสูงมากต่อเชื้อวัณโรค และไม่มีหลักฐานว่าออกฤทธิ์ต่อเชื้อแบคทีเรียชนิดอื่น

ความเชื่อมโยงระหว่างวัณโรคกับการติดเชื้อหูชั้นกลางเป็นเพียงทางอ้อม กล่าวคือ *M. tuberculosis* อาจแพร่กระจายผ่านท่อยูสเตเชียนหรือกระแสเลือดไปยังหูชั้นกลาง ก่อให้เกิด "tuberculous otitis media" ซึ่งเป็นภาวะหูนอกปอดที่พบได้น้อยมาก ในกรณีนี้ Pyrazinamide ในฐานะยาหลักของสูตร HRZE ย่อมครอบคลุมการรักษาได้

อย่างไรก็ตาม การติดเชื้อหูชั้นกลางทั่วไปที่เกิดจากแบคทีเรียที่พบบ่อย เช่น *Streptococcus pneumoniae* หรือ *Haemophilus influenzae* ไม่มีหลักฐานใดระบุว่า Pyrazinamide จะมีประสิทธิภาพ คะแนน TxGNN ที่สูงถึง 99.96% จึงน่าจะเป็นผลจากความใกล้ชิดเชิงโทโพโลยีในกราฟความรู้ (Knowledge Graph) มากกว่ากลไกการออกฤทธิ์ที่แท้จริง

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
ข้อบ่งใช้อันดับ 1 ที่ TxGNN ทำนาย (การติดเชื้อหูชั้นกลาง) ไม่มีหลักฐานจากการทดลองทางคลินิกหรือวรรณกรรมรองรับเลย และกลไกการออกฤทธิ์ของ Pyrazinamide มีความเฉพาะเจาะจงสูงต่อ *M. tuberculosis* เท่านั้น ความสมเหตุสมผลทางชีววิทยาสำหรับการใช้กับโรคหูชั้นกลางจากเชื้อทั่วไปจึงต่ำมาก

**หากต้องการดำเนินการต่อต้อง:**
- ยืนยันกลไกการออกฤทธิ์ (MOA) โดยละเอียดจาก DrugBank API (DG002)
- สืบค้นข้อมูลคำเตือนและข้อห้ามใช้จากเอกสารกำกับยา TFDA (DG001)
- พิจารณาเปลี่ยนกรอบการวิจัยจาก "infectious otitis media" ทั่วไป มาเป็น "tuberculous otitis media" ซึ่งมี Case series หลายรายการในอันดับ 2–5 (L4) รองรับ และมีความสมเหตุสมผลทางชีววิทยาสูงกว่า
- ตรวจสอบความครบถ้วนของทะเบียนยาในประเทศไทย เนื่องจากผลการสืบค้นพบว่า Pyrazinamide ยังไม่มีทะเบียนยาในไทย ซึ่งเป็นอุปสรรคสำคัญต่อการนำไปใช้ทางคลินิก
