# Ceftriaxone: จากการติดเชื้อแบคทีเรียสู่ภาวะอะไมเลสในเลือดสูง

## สรุปสั้นๆ

Ceftriaxone เป็นยาปฏิชีวนะกลุ่ม Third-generation Cephalosporin ที่ใช้รักษาโรคติดเชื้อแบคทีเรียชนิดต่างๆ ได้แก่ เยื่อหุ้มสมองอักเสบ ปอดอักเสบ การติดเชื้อในกระแสเลือด และการติดเชื้อทางเดินปัสสาวะ
โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **ภาวะอะไมเลสในเลือดสูง (Hyperamylasemia)** ซึ่งเป็นภาวะที่ระดับเอนไซม์อะไมเลสในเลือดสูงเกินปกติ
ปัจจุบันมีเพียง **วรรณกรรม 3 ฉบับ** ที่เกี่ยวข้อง ไม่มีการทดลองทางคลินิกโดยตรง และหลักฐานชี้ว่าความสัมพันธ์นี้เป็นแบบทางอ้อม ไม่ใช่การรักษาภาวะอะไมเลสสูงโดยตรง

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์อย่างเป็นทางการจากฐานข้อมูล DrugBank อย่างไรก็ตาม จากความรู้ทางเภสัชวิทยาทั่วไป Ceftriaxone ออกฤทธิ์โดยจับกับ Penicillin-Binding Proteins (PBPs) บนผนังเซลล์แบคทีเรีย ยับยั้งการสังเคราะห์ peptidoglycan จนเซลล์แบคทีเรียแตกสลาย มีฤทธิ์ครอบคลุมแบคทีเรียทั้ง Gram-positive และ Gram-negative หลายชนิด รวมถึงมีค่าครึ่งชีวิตยาวประมาณ 6–8 ชั่วโมง เหมาะสำหรับการให้วันละครั้ง

ความเชื่อมโยงระหว่าง Ceftriaxone กับภาวะอะไมเลสในเลือดสูงนั้น **เป็นแบบทางอ้อม** กล่าวคือ ยาสามารถรักษาการติดเชื้อแบคทีเรียที่เป็นสาเหตุต้นน้ำของภาวะอะไมเลสสูง เช่น การติดเชื้อ Leptospira (Weil syndrome) ซึ่งอาจทำให้เกิดการอักเสบหลายอวัยวะรวมถึงตับอ่อน หรือการติดเชื้อรุนแรงที่นำไปสู่ hyperamylasemia รอง อย่างไรก็ตาม ยังไม่มีหลักฐานใดที่แสดงว่า Ceftriaxone ลดระดับอะไมเลสในเลือดได้โดยตรง

การคาดการณ์จาก TxGNN น่าจะสะท้อนความสัมพันธ์เชิงเครือข่ายในกราฟความรู้ (Knowledge Graph) ระหว่างยาต้านแบคทีเรียกลุ่ม Cephalosporin กับภาวะที่เกิดจากการติดเชื้อรุนแรง มากกว่าจะชี้ถึงกลไกเภสัชวิทยาเฉพาะเจาะจง ดังนั้นจำเป็นต้องยืนยันด้วยการศึกษาทางคลินิกที่ออกแบบมาโดยเฉพาะก่อนสรุป

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
หลักฐานที่มีอยู่เป็นเพียงรายงานผู้ป่วยและการศึกษาเชิงสังเกต (L4) ซึ่งไม่ได้ออกแบบมาเพื่อประเมินประสิทธิผลของ Ceftriaxone ต่อภาวะอะไมเลสสูงโดยตรง ความสัมพันธ์เป็นแบบทางอ้อม และยาไม่ได้ขึ้นทะเบียนในประเทศไทย จึงยังไม่สมควรดำเนินการต่อในขั้นนี้

**หากต้องการดำเนินการต่อต้อง:**
- ดาวน์โหลดและวิเคราะห์ฉลากยา Ceftriaxone จาก Thai FDA เพื่อประเมินข้อห้ามใช้และคำเตือน (แก้ไข Data Gap DG001)
- ยืนยันกลไกการออกฤทธิ์ (MOA) จาก DrugBank API (แก้ไข Data Gap DG002)
- ระบุสาเหตุเฉพาะของ hyperamylasemia ที่ Ceftriaxone อาจรักษาได้ทางอ้อม เช่น leptospirosis-associated hyperamylasemia หรือ post-ERCP hyperamylasemia
- ออกแบบโปรโตคอลศึกษาเพื่อพิสูจน์ผลของ Ceftriaxone ต่อระดับอะไมเลสในกรณีเฉพาะก่อนพิจารณาขั้นต่อไป
