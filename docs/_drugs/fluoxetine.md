# Fluoxetine: จากยาต้านซึมเศร้า (SSRI) สู่โรคบุคลิกภาพแบบ Schizoid

## สรุปสั้นๆ

Fluoxetine เป็นยากลุ่ม Selective Serotonin Reuptake Inhibitor (SSRI) ที่ใช้รักษาโรคซึมเศร้า โรควิตกกังวล และโรคย้ำคิดย้ำทำมาอย่างยาวนาน โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **โรคบุคลิกภาพแบบ Schizoid (Schizoid Personality Disorder)** ซึ่งเป็นโรคบุคลิกภาพกลุ่ม A ที่มีลักษณะเด่นคืออารมณ์เย็นชาและหลีกเลี่ยงสังคม ปัจจุบันมีเพียง **วรรณกรรม 3 ฉบับ** ที่เกี่ยวข้องโดยอ้อม และ **ไม่มีการทดลองทางคลินิกที่ลงทะเบียนไว้** สนับสนุนแนวทางนี้

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียดจากฐานข้อมูล DrugBank อย่างไรก็ตาม Fluoxetine เป็นที่ทราบกันดีว่าออกฤทธิ์โดยการยับยั้ง Serotonin Transporter (SERT) ทำให้ระดับเซโรโทนินในไซแนปส์เพิ่มสูงขึ้น กลไกนี้ได้รับการพิสูจน์แล้วในการรักษาโรคซึมเศร้า โรควิตกกังวล และโรคย้ำคิดย้ำทำ

โรคบุคลิกภาพแบบ Schizoid มีอาการหลัก ได้แก่ อารมณ์แบน (emotional flatness) ภาวะไม่มีความสุข (anhedonia) และการหลีกเลี่ยงความสัมพันธ์ทางสังคม ซึ่งยังไม่มีการพิสูจน์ทางชีววิทยาว่าเชื่อมโยงกับวิถีเซโรโทนินโดยตรง วรรณกรรมที่มีอยู่บ่งชี้เพียงว่า SSRIs อาจมีประโยชน์บ้างในโรคบุคลิกภาพกลุ่ม A โดยรวม แต่ยังขาดการศึกษาที่ออกแบบมาสำหรับ Schizoid PD โดยเฉพาะ

การคาดการณ์ของ TxGNN จึงอิงจากรูปแบบเครือข่ายในกราฟความรู้ (Knowledge Graph) เป็นหลัก โดยไม่มีหลักฐานทางคลินิกโดยตรงรองรับ ความเชื่อมโยงทางกลไกจัดอยู่ในระดับการอนุมาน (speculative) เท่านั้น

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
หลักฐานอยู่ในระดับ L4 มีเพียงวรรณกรรม 3 ฉบับที่เกี่ยวข้องโดยอ้อม ไม่มีการทดลองทางคลินิกใดที่ประเมิน Fluoxetine สำหรับโรคบุคลิกภาพแบบ Schizoid โดยตรง และความเชื่อมโยงทางกลไกระหว่าง SERT inhibition กับอาการหลักของโรคนี้ยังไม่มีการพิสูจน์ทางชีววิทยา

**หากต้องการดำเนินการต่อต้อง:**
- ดาวน์โหลดและวิเคราะห์เอกสารกำกับยาจาก Thai FDA เพื่อรับข้อมูลคำเตือนและข้อห้ามใช้ (Data Gap: DG001 — ระดับความเร่งด่วน Blocking)
- สืบค้นข้อมูลกลไกการออกฤทธิ์โดยละเอียด (MOA) จาก DrugBank API (Data Gap: DG002 — ระดับความเร่งด่วน High)
- ออกแบบการศึกษาก่อนคลินิก (in vitro / in vivo) เพื่อประเมินผลของ SERT inhibition ต่ออาการ schizoid เพื่อยกระดับหลักฐานจาก L4 → L3
- ขยายขอบเขตการทบทวนวรรณกรรมไปยัง SSRIs ทั้งกลุ่มในโรคบุคลิกภาพกลุ่ม A อย่างเป็นระบบ
