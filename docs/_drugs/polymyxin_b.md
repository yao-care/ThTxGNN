# Polymyxin B (多黏菌素B) - 藥師評估報告

## 一句話總結

<p class="key-answer" data-question="Polymyxin B 可以用於治療什麼新適應症？">
多黏菌素B是一種抗革蘭氏陰性菌的多肽類抗生素,TxGNN預測其可用於支氣管炎和結膜炎,其中結膜炎已有Phase 3/4臨床試驗支持,支氣管炎則有豐富的文獻證據支持吸入式治療。
</p>

<h3>為什麼這個預測合理？</h3>

<p>### 支氣管炎預測分析 (TxGNN Score: 0.999, Rank: 3514)</p>

<ol>
<li><strong>抗菌機制</strong>: Polymyxin B對革蘭氏陰性菌(如綠膿桿菌、鮑氏不動桿菌)具強效殺菌作用</li>
<li><strong>呼吸道感染關聯</strong>: 這些病原菌是呼吸機相關肺炎和氣管支氣管炎的主要致病菌</li>
<li><strong>吸入給藥經驗</strong>: 已有吸入式polymyxin B用於多重抗藥性菌感染的臨床經驗</li>
<li><strong>抗藥性考量</strong>: 作為最後防線抗生素,對多重抗藥性菌株有效</li>
</ol>

## 泰國上市資訊

### 現有核准製劑

泰國已有多種含Polymyxin B的眼用複方製劑,主要適應症為:
- 細菌性眼部感染
- 急慢性結膜炎
- 角膜炎、角膜潰瘍
- 眼瞼炎、淚囊炎

### 劑型分布
- 眼藥膏/眼藥水 (主要)
- 外用軟膏
- 耳用製劑

## 安全性考量

### 已知安全性資訊

1. **腎毒性**: 全身性使用有腎毒性風險,需監測腎功能
2. **神經毒性**: 可能引起神經肌肉阻斷
3. **過敏反應**: 吸入給藥可能引起支氣管痙攣
4. **局部刺激**: 眼用製劑可能引起短暫刺激

### 吸入給藥特殊考量

1. **支氣管痙攣風險**: 文獻報告吸入後可能誘發支氣管收縮
2. **霧化前處理**: 建議先使用支氣管擴張劑
3. **監測需求**: 需密切監測呼吸功能

### 藥物交互作用

DDI資料庫未記載顯著交互作用,但應注意:
- 與其他腎毒性藥物併用需謹慎
- 與神經肌肉阻斷劑可能有加成作用

## 結論與下一步

### 整體評估

| 評估項目 | 支氣管炎 | 結膜炎 |
|----------|----------|--------|
| 證據等級 | L3 (觀察性研究) | L2 (Phase 3/4) |
| 機制合理性 | 高 | 高 (已核准) |
| 臨床可行性 | 中 (限特定患者群) | 高 |
| 建議優先順序 | 中 | 已實現 |

### 建議

**支氣管炎適應症**:
1. **適用族群**: 多重抗藥性革蘭氏陰性菌引起的呼吸道感染
2. **給藥途徑**: 吸入式霧化給藥
3. **進一步研究**: 建議進行前瞻性隨機對照試驗
4. **臨床定位**: 作為多重抗藥性感染的救援性治療選項

**結膜炎適應症**:
- 已有充分證據支持,目前核准適應症涵蓋此用途

### 限制說明

- 吸入式給藥在泰國尚無專用製劑
- 支氣管炎文獻多為救援性治療,非第一線用藥
- 需考慮抗藥性管理策略
