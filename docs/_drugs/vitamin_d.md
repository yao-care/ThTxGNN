# Vitamin D: จากสุขภาพกระดูกและแคลเซียมสู่ความผิดปกติการปล่อยเกล็ดเลือดปฐมภูมิ

## สรุปสั้นๆ

Vitamin D (DrugBank ID: DB11094) เป็นวิตามินที่ละลายในไขมัน มีบทบาทหลักในการควบคุมการดูดซึมแคลเซียมและฟอสฟอรัส ดูแลสุขภาพกระดูก และรักษาภาวะขาดวิตามินดี
โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **ความผิดปกติการปล่อยเกล็ดเลือดปฐมภูมิ (Primary Release Disorder of Platelets)** ด้วยคะแนน 93.11%
อย่างไรก็ตาม จากการค้นหาพบ **การทดลองทางคลินิก 11 รายการ** แต่ **ไม่มีรายการใดมีความเกี่ยวข้องโดยตรง** กับเป้าหมายการรักษา และ **ไม่พบวรรณกรรมสนับสนุน** การใช้งานนี้เลย

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ตัวรับวิตามินดี (Vitamin D Receptor, VDR) มีการแสดงออกในเมกะคาริโอไซต์ (megakaryocytes) ซึ่งเป็นเซลล์ต้นกำเนิดของเกล็ดเลือด และในเกล็ดเลือดโดยตรง ในทางทฤษฎี การกระตุ้น VDR อาจมีส่วนในการปรับสัญญาณที่ควบคุมกระบวนการปล่อยสารจากแกรนูลของเกล็ดเลือด (platelet granule release signaling pathway) ซึ่งเป็นรากฐานที่ทำให้โมเดล TxGNN คาดการณ์ความเชื่อมโยงนี้

อย่างไรก็ตาม การเชื่อมโยงเชิงกลไกระหว่าง Vitamin D กับความผิดปกติการปล่อยเกล็ดเลือดปฐมภูมิโดยเฉพาะ ถือว่ามีระดับการคาดเดาสูงมาก ความผิดปกตินี้ส่วนใหญ่เกิดจากความบกพร่องของเส้นทางสัญญาณเฉพาะเจาะจงในเกล็ดเลือด (เช่น cyclooxygenase, ADP pathway, หรือ storage pool deficiency) ซึ่งไม่ได้อยู่ภายใต้การควบคุมหลักของ VDR

ปัจจุบันยังไม่มีหลักฐานจากการทดลองทางคลินิก งานวิจัยก่อนคลินิก หรือวรรณกรรมใดๆ ที่ยืนยันว่า Vitamin D สามารถแก้ไขหรือบรรเทาความผิดปกติประเภทนี้ได้ การทดลองที่ค้นพบจากการค้นหาล้วนเป็นหัวข้ออื่นที่ไม่เกี่ยวข้อง

## ข้อมูลการวางจำหน่ายในประเทศไทย

ปัจจุบัน Vitamin D (DrugBank ID: DB11094) **ยังไม่มีทะเบียนยาในระบบสำนักงานคณะกรรมการอาหารและยา (อย.) ของประเทศไทย** ไม่พบใบอนุญาตใดๆ ในฐานข้อมูลที่ตรวจสอบ

> หมายเหตุ: ผลิตภัณฑ์วิตามินดีในรูปแบบอาหารเสริม (supplement) อาจวางจำหน่ายในประเทศไทยภายใต้การกำกับดูแลของ อย. ในหมวดผลิตภัณฑ์เสริมอาหาร ซึ่งต่างจากการจดทะเบียนยาแผนปัจจุบัน

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
หลักฐานปัจจุบันอยู่ในระดับ L5 ซึ่งหมายความว่ามีเพียงการทำนายจากโมเดล TxGNN เท่านั้น ไม่มีหลักฐานทางคลินิกหรือก่อนคลินิกใดๆ ที่ยืนยันผลของ Vitamin D ต่อความผิดปกติการปล่อยเกล็ดเลือดปฐมภูมิ การเชื่อมโยงเชิงกลไกที่เสนอผ่าน VDR ยังคงเป็นการอนุมานทางทฤษฎีสูง

**หากต้องการดำเนินการต่อต้อง:**
- ดำเนินการศึกษาพื้นฐาน (in vitro/in vivo) เพื่อยืนยันผลของ VDR activation ต่อ platelet granule release pathway ในรูปแบบจำลองความผิดปกตินี้
- ค้นหาวรรณกรรมเพิ่มเติมในฐานข้อมูล EMBASE และ Cochrane โดยใช้คำค้นที่เฉพาะเจาะจง เช่น "vitamin D platelet function" หรือ "calcitriol megakaryocyte"
- ดาวน์โหลดเอกสารกำกับยา (วสน.) จาก TFDA เพื่อรับข้อมูลกลไกการออกฤทธิ์โดยละเอียดและข้อมูลความปลอดภัย
- พิจารณาสลับเป้าหมายไปยัง **ภาวะพาราไทรอยด์ทำงานน้อย (Hypoparathyroidism, ลำดับที่ 9)** ซึ่งมีหลักฐานระดับ L2 และคำแนะนำ **Proceed with Guardrails** พร้อมการทดลองทางคลินิก Phase 3 หลายรายการที่ตรงกับกลไกการออกฤทธิ์หลักของ Vitamin D โดยตรง
