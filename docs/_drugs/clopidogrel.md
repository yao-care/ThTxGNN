# Clopidogrel: จากยาต้านเกล็ดเลือดสู่ไมเกรนชนิดมีออร่าจากก้านสมอง

## สรุปสั้นๆ

Clopidogrel เป็นยาต้านเกล็ดเลือดกลุ่ม Thienopyridine ที่ออกฤทธิ์ผ่านการยับยั้ง P2Y12 ADP receptor ใช้ป้องกันภาวะลิ่มเลือดในผู้ป่วยโรคหลอดเลือดหัวใจเฉียบพลัน (ACS) และโรคหลอดเลือดสมอง
โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **ไมเกรนชนิดมีออร่าจากก้านสมอง (Migraine with Brainstem Aura)**
ปัจจุบันมี **การทดลองทางคลินิก 0 รายการ** ที่เฉพาะเจาะจงกับข้อบ่งใช้นี้ และ **วรรณกรรม 16 ฉบับ** ที่ส่วนใหญ่เชื่อมโยงกับกลุ่มผู้ป่วยไมเกรนที่มี Patent Foramen Ovale (PFO) สนับสนุนแนวทางนี้

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียดจาก DrugBank ตามข้อมูลที่ทราบ Clopidogrel เป็น prodrug ในกลุ่ม Thienopyridine ที่ถูกเปลี่ยนแปลงโดยเอนไซม์ CYP2C19 ในตับ ผลิตภัณฑ์ที่ออกฤทธิ์จะจับกับ P2Y12 ADP receptor บนเกล็ดเลือดแบบถาวร ลดการรวมตัวของเกล็ดเลือดและยับยั้งการก่อตัวของลิ่มเลือด ซึ่งพิสูจน์ประสิทธิผลแล้วในการรักษาโรคหลอดเลือดหัวใจและสมอง

สมมติฐานกลไกหลักที่เชื่อมโยง Clopidogrel กับไมเกรนชนิดมีออร่าจากก้านสมองคือ: ในผู้ป่วยที่มีรอยรั่วระหว่างหัวใจห้องบน เช่น Patent Foramen Ovale (PFO) หรือ Atrial Septal Defect (ASD) ไมโครเอมโบลีจากฝั่งขวาของหัวใจสามารถข้ามรอยรั่วเข้าสู่ระบบไหลเวียนสมอง กระตุ้น Cortical Spreading Depression (CSD) ซึ่งเชื่อว่าเป็นกลไกเบื้องหลังของออร่าและความปวดในไมเกรน Clopidogrel อาจช่วยลดความถี่ของไมเกรนโดยขัดขวางการสะสมของเกล็ดเลือดและลดการเกิดไมโครเอมโบลีดังกล่าว

นอกจากนี้ยังมีหลักฐานเบื้องต้นจากสัตว์ทดลองว่า P2Y12 receptor บน microglia ในนิวเคลียสหางไตรเจมินัล (Trigeminal Nucleus Caudalis) มีบทบาทในพยาธิกำเนิดของไมเกรนเรื้อรังผ่านเส้นทาง RhoA/ROCK ซึ่งหาก Clopidogrel สามารถยับยั้งการกระตุ้น microglia นี้ได้ จะถือเป็น dual mechanism ที่น่าสนใจ อย่างไรก็ตาม หลักฐานที่มีส่วนใหญ่เกี่ยวข้องกับไมเกรนที่มี PFO/ASD เป็นพื้นฐาน และยังไม่มีการทดลองที่เจาะจงสำหรับ "ออร่าจากก้านสมอง" โดยตรง

ปัจจุบันยังไม่มีการลงทะเบียนการทดลองทางคลินิกที่เกี่ยวข้องกับ Clopidogrel สำหรับข้อบ่งใช้ไมเกรนชนิดมีออร่าจากก้านสมองโดยตรง

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
หลักฐานที่มีอยู่เชื่อมโยง Clopidogrel กับการลดไมเกรนในบริบทของผู้ป่วยที่มี PFO/ASD เป็นหลัก แต่สำหรับ "ไมเกรนชนิดมีออร่าจากก้านสมอง" โดยเฉพาะเจาะจงนั้น ยังไม่มีการทดลองทางคลินิกที่กำหนดกลุ่มประชากรนี้ไว้อย่างชัดเจน และระดับหลักฐานที่ L3 ยังต้องการการวิจัยเพิ่มเติมก่อนดำเนินการต่อ

**หากต้องการดำเนินการต่อต้อง:**
- ออกแบบ Proof-of-concept trial ในกลุ่มผู้ป่วยไมเกรน brainstem aura ที่ได้รับการยืนยัน PFO ด้วย contrast-enhanced TCD หรือ echocardiography
- เติมข้อมูล MOA โดยละเอียดผ่าน DrugBank API (แก้ไข Data Gap DG002)
- ดาวน์โหลดและวิเคราะห์คำเตือน ข้อห้ามใช้ และปฏิกิริยาระหว่างยาจากเอกสารกำกับยา TFDA (แก้ไข Data Gap DG001)
- ตรวจสอบว่า clopidogrel ขึ้นทะเบียนในฐานข้อมูล TFDA ในชื่ออื่น (เช่น Plavix® หรือชื่อสามัญภาษาไทย) เนื่องจากผลการสืบค้น 0 รายการอาจเป็นปัญหาด้านการค้นหา
