# Colchicine: จากเก๊าท์สู่มาลาเรีย Plasmodium falciparum

## สรุปสั้นๆ

Colchicine เป็นยาคลาสสิกที่ใช้รักษาโรคเก๊าท์และ Familial Mediterranean Fever (FMF) มาหลายทศวรรษ โดยมีกลไกหลักที่เกี่ยวข้องกับการยับยั้งการรวมตัวของ β-tubulin และการกดการอักเสบ
โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **มาลาเรีย Plasmodium falciparum (Plasmodium falciparum malaria)**
อย่างไรก็ตาม ปัจจุบัน **ยังไม่มีการทดลองทางคลินิกหรือวรรณกรรมใดรองรับ** แนวทางนี้โดยตรง การทำนายนี้อ้างอิงจากความใกล้เคียงในเครือข่ายความรู้ของโมเดลเท่านั้น

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียดในชุดหลักฐานนี้ ตามที่ทราบโดยทั่วไป Colchicine ออกฤทธิ์โดยการยับยั้งการรวมตัวของ β-tubulin ซึ่งรบกวนการแบ่งเซลล์แบบ mitosis และขัดขวางการเคลื่อนที่ของนิวโทรฟิล (neutrophil chemotaxis) นอกจากนี้ยังมีผลยับยั้ง NLRP3 inflammasome และการหลั่ง IL-1β ด้วย

ในเชิงทฤษฎี กลไกการยับยั้ง tubulin อาจส่งผลต่อ *P. falciparum* ในช่วง intraerythrocytic stage เนื่องจากเชื้อมาลาเรียต้องอาศัยโปรตีน tubulin ในกระบวนการแบ่งเซลล์ภายในเม็ดเลือดแดงเช่นกัน ความเชื่อมโยงระหว่างโรคเก๊าท์/FMF กับมาลาเรียไม่ได้อยู่ที่กลุ่มโรค แต่อยู่ที่กลไกต้านการแบ่งเซลล์ร่วมกัน

อย่างไรก็ตาม การอนุมานเชิงกลไกนี้**ยังไม่ได้รับการสนับสนุนจากการศึกษาใดๆ** ทั้งในระดับก่อนคลินิกและคลินิก การทำนายนี้มาจากความใกล้เคียงในเครือข่ายความรู้ของ TxGNN เท่านั้น ไม่ควรนำไปใช้ตัดสินใจทางคลินิกโดยไม่มีหลักฐานเพิ่มเติม

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
การทำนายนี้มีเพียงคะแนนจากโมเดล TxGNN โดยไม่มีหลักฐานจากการทดลองทางคลินิกหรือวรรณกรรมรองรับเลย (ระดับ L5) และยาอยู่ในสถานะยังไม่วางจำหน่ายในไทย ทำให้ยังไม่เหมาะสมที่จะดำเนินการต่อในขณะนี้

**หากต้องการดำเนินการต่อต้อง:**
- ศึกษากลไกการออกฤทธิ์ของ Colchicine ต่อ *P. falciparum* ในระดับก่อนคลินิก (in vitro และ in vivo) โดยเฉพาะผลต่อ tubulin ของเชื้อ
- เติมข้อมูลกลไกการออกฤทธิ์ (MOA) จาก DrugBank API ซึ่งปัจจุบันยังเป็น Data Gap
- ดาวน์โหลดและวิเคราะห์ข้อมูลความปลอดภัย/ข้อห้ามใช้จากเอกสารกำกับยา TFDA
- ตรวจสอบสถานะการขึ้นทะเบียนและเส้นทางการนำเข้าสำหรับประเทศไทย เนื่องจากยาไม่มีใบอนุญาตในปัจจุบัน
- พิจารณา indication ที่มีหลักฐานแข็งแกร่งกว่า เช่น **Familial Mediterranean Fever (rank 2, L3, Proceed with Guardrails)** ซึ่งมีวรรณกรรมรองรับ 20 ฉบับและ Colchicine เป็นยามาตรฐานที่ใช้อยู่แล้วในระดับสากล
