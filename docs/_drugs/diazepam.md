# Diazepam: จากยาต้านวิตกกังวลและยากันชักสู่การรักษาโรคนอนไม่หลับ

## สรุปสั้นๆ

Diazepam เป็นยาในกลุ่มเบนโซไดอาซีพีน ซึ่งมีการใช้งานอย่างแพร่หลายทั่วโลกในการรักษาโรควิตกกังวล อาการชัก อาการเกร็งของกล้ามเนื้อ และภาวะถอนพิษแอลกอฮอล์ โมเดล TxGNN คาดการณ์ว่า Diazepam อาจมีศักยภาพในการรักษา **โรคนอนไม่หลับ (Insomnia)** ด้วยคะแนนการทำนายสูงสุดถึง 99.999% ปัจจุบันมี **การทดลองทางคลินิก 24 รายการ** และ **วรรณกรรม 18 ฉบับ** ที่เกี่ยวข้องในฐานข้อมูล โดยมีหลักฐาน RCT โดยตรง 1 รายการที่เปรียบเทียบประสิทธิผลของ Diazepam กับยาอื่นในผู้ป่วยโรคนอนไม่หลับ

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียดจากฐานข้อมูล DrugBank อย่างไรก็ตาม จากวรรณกรรมทางวิทยาศาสตร์ (PMID 39581171) ทราบว่า Diazepam ทำหน้าที่เป็น **Positive Allosteric Modulator (PAM)** ของ GABA-A Receptor โดยเพิ่มความถี่การเปิดของช่อง Chloride Ion ส่งผลให้การยับยั้งการทำงานของระบบประสาทส่วนกลางเพิ่มขึ้น กลไกนี้นำไปสู่ฤทธิ์ทางเภสัชวิทยาหลายประการ ได้แก่ ลดความวิตกกังวล คลายกล้ามเนื้อ ต้านอาการชัก และทำให้ง่วงนอน

การนอนหลับของมนุษย์ถูกควบคุมอย่างสำคัญโดยระบบ GABAergic ซึ่งหมายความว่า Diazepam สามารถออกฤทธิ์โดยตรงต่อวงจรประสาทที่ควบคุมการตื่น-นอน ผลคือ **การลดระยะเวลาเริ่มต้นนอนหลับ (Sleep Onset Latency)** และ **ลดจำนวนครั้งที่ตื่นระหว่างคืน** ซึ่งตรงกับอาการหลักของโรคนอนไม่หลับ หลักฐาน RCT (PMID 6113175) ที่เปรียบเทียบ Diazepam 5 mg กับ Lormetazepam ในผู้ป่วยนอนไม่หลับยืนยันว่า Diazepam มีผลจริงต่อการลดเวลาเริ่มหลับและยืดระยะเวลานอนต่อเนื่อง

อย่างไรก็ตาม ควรระวังข้อจำกัดสำคัญ: การใช้ Diazepam ระยะยาวมีความเสี่ยงต่อการยับยั้ง Slow Wave Sleep (SWS) และ REM Sleep ทำให้คุณภาพการนอนหลับลดลงในระยะยาว รวมถึงอาจเกิดการพึ่งพาทางกายภาพและจิตใจ งานวิจัยใน Nature Neuroscience (PMID 35228700) ยังพบว่าการใช้ Diazepam ระยะยาวอาจก่อให้เกิดความบกพร่องทางสติปัญญาผ่านกลไก TSPO และ Microglial Dendritic Spine Engulfment ดังนั้นการนำมาใช้ในโรคนอนไม่หลับจึงควรจำกัดเฉพาะระยะสั้นและมีการควบคุมดูแลอย่างเข้มงวด

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Proceed with Guardrails**

**เหตุผล:**
มีหลักฐาน RCT โดยตรง (PMID 6113175) ยืนยันว่า Diazepam มีฤทธิ์ช่วยให้นอนหลับในผู้ป่วยโรคนอนไม่หลับ บวกกับกลไก GABA-A PAM ที่ชัดเจนและการทดลองทางคลินิก Phase 3 ที่กำลังดำเนินการสำหรับ BZD class ในบริบทโรคนอนไม่หลับ อย่างไรก็ตาม ความเสี่ยงต่อการพึ่งพา การยับยั้ง SWS/REM sleep ความบกพร่องทางสติปัญญาระยะยาว และการขาดข้อมูลการจดทะเบียนใน TFDA จำเป็นต้องมีการวางแผนและควบคุมดูแลอย่างเข้มงวดก่อนดำเนินการต่อ

**หากต้องการดำเนินการต่อต้อง:**
- ตรวจสอบสถานะการจดทะเบียน Diazepam กับ อย. อย่างละเอียด เนื่องจากอาจเป็นยาควบคุมพิเศษ (Schedule IV) ที่ไม่ปรากฏในฐานข้อมูล TFDA สาธารณะ
- ดาวน์โหลดและวิเคราะห์เอกสารกำกับยา (Package Insert) จาก TFDA เพื่อรับข้อมูลคำเตือน ข้อห้ามใช้ และปฏิกิริยาระหว่างยา
- สืบค้น DrugBank API เพิ่มเติมเพื่อรับข้อมูล MOA และ DDI ที่ครบถ้วน
- กำหนดโปรโตคอลการใช้ยาระยะสั้น (แนะนำไม่เกิน 2–4 สัปดาห์) พร้อมแผนการลดขนาดยาอย่างค่อยเป็นค่อยไปเพื่อหลีกเลี่ยงอาการถอนยา
- ประเมินความเสี่ยงเป็นพิเศษในกลุ่มผู้สูงอายุ ผู้ที่มีประวัติติดสารเสพติด และผู้ที่ใช้ยากดระบบประสาทส่วนกลางร่วมกัน
