# Levothyroxine: จากภาวะพร่องไทรอยด์ฮอร์โมนสู่คอพอกเฉพาะถิ่น

## สรุปสั้นๆ

Levothyroxine (LT4) เป็นยาฮอร์โมนไทรอยด์สังเคราะห์ที่ใช้รักษาภาวะพร่องไทรอยด์ฮอร์โมนและทดแทนฮอร์โมนหลังผ่าตัดต่อมไทรอยด์ โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **คอพอกเฉพาะถิ่น (Endemic Goiter)** โดยอาศัยกลไกยับยั้ง TSH เพื่อลดการกระตุ้นให้ต่อมไทรอยด์ขยายตัว ปัจจุบันมี **การทดลองทางคลินิก 1 รายการ** และ **วรรณกรรม 20 ฉบับ** ที่เกี่ยวข้องกับบริบทนี้ แม้วรรณกรรมส่วนใหญ่จะเน้นการเสริมไอโอดีนเป็นการรักษาหลักมากกว่า LT4

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียดจากฐานข้อมูลที่ทำการสืบค้น อย่างไรก็ตาม จากหลักฐานทางคลินิกที่ทราบกันดี Levothyroxine เป็นฮอร์โมนไทรอยด์สังเคราะห์ชนิด T4 ที่เมื่อให้จากภายนอกร่างกายจะกระตุ้น negative feedback loop ของแกน hypothalamus-pituitary-thyroid ทำให้ต่อมใต้สมองลดการหลั่ง TSH ลง

คอพอกเฉพาะถิ่นเกิดจากภาวะขาดไอโอดีนเรื้อรัง ส่งผลให้ TSH สูงขึ้นต่อเนื่องและกระตุ้นเซลล์ไทรอยด์ให้แบ่งตัวจนเกิดการขยายตัวของต่อมแบบชดเชย LT4 ภายนอกสามารถตัดวงจรนี้ได้โดยการลด TSH ผ่าน negative feedback ลดสัญญาณกระตุ้นการเจริญเติบโตของเนื้อเยื่อไทรอยด์ มีรายงานจากปี 1988 (PMID 3278876) ที่ทดสอบ LT4 150 µg/วัน เปรียบกับ LT4 + potassium iodide ในผู้ป่วยคอพอกเฉพาะถิ่น 74 ราย พบว่าทั้งสองกลุ่มลดขนาดคอพอกได้ภายใน 6 เดือน นับเป็นหลักฐานโดยตรงที่ชัดเจนที่สุดที่มีในขณะนี้

อย่างไรก็ตาม การรักษาขั้นต้นตามแนวทางสากลสำหรับคอพอกเฉพาะถิ่นยังคงเป็นการเสริมไอโอดีน (เช่น เกลือเสริมไอโอดีน น้ำมันไอโอดีน) LT4 มีบทบาทสนับสนุนเป็นหลักในรายที่มีภาวะ hypothyroidism ร่วมด้วย และยังขาด RCT คุณภาพสูงในยุคปัจจุบันที่สนับสนุนการใช้ LT4 เป็นการรักษาหลักในข้อบ่งใช้นี้

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
แม้จะมีพื้นฐานทางกลไกที่สมเหตุสมผล (LT4 ลด TSH → ลดสัญญาณกระตุ้นการขยายตัวของไทรอยด์) และมีงานศึกษาโดยตรงจากปี 1988 สนับสนุน แต่การรักษามาตรฐานสำหรับคอพอกเฉพาะถิ่นยังคงเป็นการเสริมไอโอดีน ขาด RCT คุณภาพสูงในยุคปัจจุบันที่สนับสนุน LT4 เป็นการรักษาหลัก และยาชนิดนี้ยังไม่ขึ้นทะเบียนในประเทศไทย

**หากต้องการดำเนินการต่อต้อง:**
- รวบรวมข้อมูล MOA โดยละเอียดจาก DrugBank API
- ดาวน์โหลดและวิเคราะห์เอกสารกำกับยาของ Thai FDA เพื่อระบุคำเตือน ข้อห้ามใช้ และปฏิกิริยาระหว่างยา
- ศึกษา RCT สมัยใหม่ที่เปรียบเทียบ LT4 กับการเสริมไอโอดีนในพื้นที่คอพอกเฉพาะถิ่นโดยตรง
- ดำเนินการขึ้นทะเบียนยากับ Thai FDA ก่อนพิจารณานำไปใช้ในคลินิก
