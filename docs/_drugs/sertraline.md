# Sertraline: จากยาต้านซึมเศร้าและโรควิตกกังวลสู่โรคบุคลิกภาพสคิโซไทปอล

## สรุปสั้นๆ

Sertraline เป็น Selective Serotonin Reuptake Inhibitor (SSRI) ที่ใช้รักษาภาวะซึมเศร้า โรคย้ำคิดย้ำทำ และโรควิตกกังวลหลายรูปแบบในระดับสากล โดยยานี้ยังไม่ปรากฏในฐานข้อมูลการขึ้นทะเบียนของไทย

โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **โรคบุคลิกภาพสคิโซไทปอล (Schizotypal Personality Disorder)** โดยมีคะแนนทำนายสูงถึง **99.93%**

ปัจจุบันมี **การทดลองทางคลินิก 1 รายการ** และ **วรรณกรรม 1 ฉบับ** ที่เกี่ยวข้อง ซึ่งถือว่าหลักฐานยังอยู่ในระดับที่ต้องการการศึกษาเพิ่มเติมอย่างมีนัยสำคัญ

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียดจากฐานข้อมูล DrugBank อย่างไรก็ตามในฐานะ SSRI Sertraline ออกฤทธิ์โดยยับยั้ง Serotonin Transporter (SERT) ทำให้เซโรโทนินสะสมในช่องไซแนปส์และปรับสมดุลของวงจรอารมณ์ ความวิตกกังวล และการประมวลผลสัญญาณรางวัลในสมองส่วนลิมบิกและคอร์เท็กซ์ส่วนหน้า

ความเชื่อมโยงทางกลไกกับโรคบุคลิกภาพสคิโซไทปอลตั้งอยู่บนสมมติฐานว่าระบบเซโรโทนินมีบทบาทต่อความผิดปกติทางการรับรู้-ความคิด (cognitive-perceptual distortions) ซึ่งเป็นอาการหลักของโรค นอกจากนี้ Sertraline ยังอาจช่วยบรรเทาความวิตกกังวลทางสังคม ความคิดแบบ magical thinking ที่รุนแรงน้อย รวมถึงอาการซึมเศร้าที่มักพบร่วมในผู้ป่วยกลุ่มนี้

อย่างไรก็ตาม หลักฐานที่มีอยู่เพียงการทดลองนำร่องขนาดเล็กมาก (n=8) ที่ใช้ Sertraline ร่วมกับ Risperidone ทำให้ไม่สามารถประเมินผลของ Sertraline เดี่ยวต่อโรคบุคลิกภาพสคิโซไทปอลได้ และวรรณกรรมที่สืบค้นได้ไม่มีความเกี่ยวข้องโดยตรงกับโรคนี้

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
แม้คะแนน TxGNN จะสูงถึง 99.93% แต่หลักฐานทางคลินิกยังอ่อนมาก — การทดลองทางคลินิกที่มีเพียงรายการเดียวมีกลุ่มตัวอย่างเพียง 8 คน และวรรณกรรมที่สืบค้นได้ไม่มีความเกี่ยวข้องโดยตรงกับ schizotypal personality disorder การทำนายในขณะนี้อยู่บนพื้นฐานของการอนุมานทางกลไกและ knowledge graph เป็นหลัก ยังไม่เพียงพอสำหรับการดำเนินการเชิงคลินิก

**หากต้องการดำเนินการต่อต้อง:**
- ดาวน์โหลดและวิเคราะห์เอกสารกำกับยา (SmPC/Package Insert) เพื่อเติมข้อมูลความปลอดภัย คำเตือน และข้อห้ามใช้
- สืบค้นข้อมูล MOA จาก DrugBank API อย่างครบถ้วนเพื่อเสริมการวิเคราะห์กลไก
- สำรวจการศึกษาเชิงสังเกต (observational studies) เพิ่มเติมในฐานข้อมูล EMBASE/PsycINFO
- ออกแบบการทดลองนำร่อง (pilot RCT) ขนาดตัวอย่างที่เหมาะสมก่อนเลื่อนระดับหลักฐาน
- ตรวจสอบสถานะการขึ้นทะเบียนยา Sertraline ในไทยกับ อย. โดยตรง เนื่องจากอาจมีข้อมูลที่ไม่ครบในฐานข้อมูลที่ใช้สืบค้น
