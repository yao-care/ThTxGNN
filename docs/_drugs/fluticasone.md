# Fluticasone: จากโรคจมูกอักเสบจากภูมิแพ้สู่โรคปวดหัวไมเกรน

## สรุปสั้นๆ

Fluticasone เป็นยากลุ่มกลูโคคอร์ติคอยด์สังเคราะห์ที่ใช้แพร่หลายในระดับสากลสำหรับโรคจมูกอักเสบจากภูมิแพ้ หอบหืด และภาวะอักเสบของระบบทางเดินหายใจ แต่ยังไม่ได้รับการขึ้นทะเบียนในประเทศไทย
โมเดล TxGNN คาดการณ์ว่าอาจมีผลต่อ **โรคปวดหัวไมเกรน (migraine disorder)** ด้วยคะแนนทำนายสูงถึง 99.20%
อย่างไรก็ตาม ปัจจุบัน **ไม่มีการทดลองทางคลินิก** และ **วรรณกรรมที่พบมีเพียง 1 ฉบับ** ซึ่งรายงานผลข้างเคียงทางระบบประสาทจากการใช้ยา ไม่ใช่ประสิทธิผลในการรักษาไมเกรน

## ทำไมการคาดการณ์นี้จึงสมเหตุสมผล?

ปัจจุบันยังขาดข้อมูลกลไกการออกฤทธิ์โดยละเอียดจากฐานข้อมูล DrugBank ตามที่ทราบ Fluticasone เป็นยากลูโคคอร์ติคอยด์ที่ออกฤทธิ์จับกับตัวรับ glucocorticoid receptor (GR) ยับยั้งการสังเคราะห์และปลดปล่อยสารสื่ออักเสบ เช่น ไซโตไคน์ (IL-4, IL-5) พรอสตาแกลนดิน และลิวโคไตรอีน โดยทั่วไปออกฤทธิ์เฉพาะที่สูงและมีการดูดซึมเข้าสู่ระบบน้อยมาก

ในเชิงทฤษฎี ไมเกรนมีองค์ประกอบของการอักเสบในระบบประสาท-หลอดเลือด (neurogenic inflammation) ผ่านการกระตุ้นเส้นประสาทไตรเจมินัลและการหลั่ง CGRP กลไกต้านอักเสบของกลูโคคอร์ติคอยด์อาจยับยั้งกระบวนการดังกล่าวได้ในระดับสมมติฐาน อย่างไรก็ดี รูปแบบยาที่มีอยู่ในปัจจุบัน (ยาพ่นจมูก / ยาสูดพ่น) มีการดูดซึมเข้าสู่ระบบน้อยมาก ซึ่งเป็นข้อจำกัดสำคัญในการออกฤทธิ์ต่อระบบประสาทส่วนกลางที่เกี่ยวข้องกับไมเกรน

น่าสังเกตว่าวรรณกรรมเดียวที่ค้นพบ (PMID 18681087) รายงาน **ผลข้างเคียงทางระบบประสาทและจิตเวช** จากการใช้สเตียรอยด์ทางจมูก ซึ่งเป็นสัญญาณระวังมากกว่าหลักฐานสนับสนุน และทำให้ความน่าเชื่อถือของสมมติฐานนี้อ่อนแอลงอย่างมีนัยสำคัญ

## สรุปและขั้นตอนถัดไป

**การตัดสินใจ: Hold**

**เหตุผล:**
หลักฐานปัจจุบันอยู่ในระดับ L5 (การทำนายจากโมเดลเท่านั้น) ไม่มีการทดลองทางคลินิกสนับสนุน และวรรณกรรมที่พบรายงานความเสี่ยงทางระบบประสาทแทนที่จะเป็นประสิทธิผล ประกอบกับยายังไม่ได้รับการขึ้นทะเบียนในประเทศไทย จึงยังไม่พร้อมสำหรับการดำเนินการในขั้นต่อไป

**หากต้องการดำเนินการต่อต้อง:**
- ดึงข้อมูลกลไกการออกฤทธิ์ (MOA) จาก DrugBank API เพื่อประเมินความเชื่อมโยงเชิงเภสัชวิทยา
- ดาวน์โหลดและวิเคราะห์เอกสารกำกับยา (package insert) เพื่อประเมินความปลอดภัยเบื้องต้น (S1)
- ดำเนินการค้นหาวรรณกรรมเพิ่มเติมเฉพาะด้าน glucocorticoid กับไมเกรนหรือ neurogenic inflammation
- พิจารณาออกแบบการศึกษาเชิงสังเกต (observational study) เพื่อตรวจสอบสมมติฐานก่อนเข้าสู่การทดลองทางคลินิก
- ดำเนินการขึ้นทะเบียนยากับ อย. ประเทศไทย หากต้องการวิจัยในประชากรไทย
