# ThTxGNN - ประเทศไทย: การคาดการณ์การใช้ยาเก่าในข้อบ่งใช้ใหม่

[![Website](https://img.shields.io/badge/Website-thtxgnn.yao.care-blue)](https://thtxgnn.yao.care)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

การคาดการณ์การใช้ยาเก่าในข้อบ่งใช้ใหม่ (drug repurposing) สำหรับประเทศไทยโดยใช้โมเดล TxGNN

## ข้อควรระวัง

- ผลการคาดการณ์นี้มีไว้เพื่อการวิจัยเท่านั้น ไม่ถือเป็นคำแนะนำทางการแพทย์
- การใช้ยาเก่าในข้อบ่งใช้ใหม่ต้องผ่านการทดลองทางคลินิกก่อนนำไปใช้

## วิธีการคาดการณ์

### วิธี Knowledge Graph
ค้นหาความสัมพันธ์ยา-โรคโดยตรงจาก TxGNN Knowledge Graph ระบุผู้สมัครที่มีศักยภาพสำหรับการใช้ยาเก่าในข้อบ่งใช้ใหม่จากความเชื่อมโยงที่มีอยู่ในเครือข่ายชีวการแพทย์

### วิธี Deep Learning
ใช้โมเดล Neural Network ที่ผ่านการ pretrain จาก TxGNN เพื่อคำนวณคะแนนความน่าจะเป็น ประเมินโอกาสของข้อบ่งใช้ใหม่สำหรับยาที่ได้รับอนุมัติแล้ว
